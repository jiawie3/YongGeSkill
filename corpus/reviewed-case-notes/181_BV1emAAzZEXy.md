## User Facts Extracted

- 门店形态：上海郊区的茶馆/茶空间，两层，主卖空间和茶，也附带简餐、炒菜、水饺/馄饨等家常餐食。
- 投入：总投资将近300万元。门店已开约两年。
- 经营目标：老板35岁，茶馆是副业；原来此处与文化传媒工作室有关，对场地有感情，也有自己接待需求。老板和合伙人都有主业，不完全靠茶馆赚钱，底线是“不亏钱”。
- 面积与空间：面积约450平；全包间为主，ASR 确认8个包间，另有阳台；无大厅或大厅不作为主要经营区；整店只有一个卫生间。
- 包间使用：下午喝茶、晚上可吃饭；一场限时约3小时，超时收费；有基础茶水/套餐。包间里没有麻将，老板担心影响格调，但有掼蛋等社交需求。
- 消费口径：门店口述人均约200-300元；追问平台/套餐平均时提到约120元。ASR 未确认两者分别对应哪些产品结构。
- 流水：去年平均月流水13.5万元，约4500元/日；最近几个月提升到约5000元/日。
- 毛利：整体毛利率约70%。按4500元/日流水，日毛利约3150元。
- 租金：年租金46万元，约3.83万元/月，约1277元/日。
- 人工：店内员工5人；ASR 对构成有重复/错漏，但可确认有服务员和店长。人工约3.8万-3.9万元/月，勇哥按约1300元/日估算。
- 水电气：约4500元/月，约150元/日。
- 餐食收入：餐食有人点，去年下半年到今年需求变好；老板估计按实际价格约1000多元/日，可能到1500元/日。
- 推广：目前做美团，也投了一些美团；抖音以前拍过视频但效果一般、流量下滑；小红书基本没做。
- 厨房能力：厨房不是高端餐饮厨房，厨师更像家常阿姨，菜品偏家常，认可度还可以，但出餐能力和高级感有限。
- 老板诉求：想优化利润，考虑增加品类、增加餐食或套餐，也提到想加强茶叶销售。

## Yongge Question Order

- 先问总投入，立即确认“将近300万”这个大额前期成本。
- 追问门店为什么这样开、品类是什么：确认不是纯茶饮，而是茶馆、空间、茶和简餐的复合模型。
- 问城市和区域：确认在上海郊区，判断客群和价格带。
- 问平均月流水/日流水，再问毛利率，马上换算日毛利。
- 问面积和租金，并把年租金拆到月租、日租，判断空间资产是否吃掉毛利。
- 问店内几个人、人工多少钱，再拆成日人工。
- 问水电气，再拆成日费用。
- 在日毛利、日租金、日人工、日水电都清楚后，先判断当前“赚钱但利润薄”。
- 回头问投资额、开业多久、回本情况，区分“门店经营略赚”和“投资回收困难”。
- 问年龄、主业、副业关系、合伙人关系、是否靠它赚钱，判断老板的真实目标不是快速高回报，而是接待和不亏。
- 现场看店：看前台、厨房、包间、是否有大厅、包间数量、包间是否满、棋牌/掼蛋场景、卫生间、停车、周边商场和写字楼。
- 问一场多长时间、基础套餐、人均和平台套餐均价，判断包间时长、客单价和翻台逻辑。
- 问现在做哪些推广，再问餐食每天卖多少，判断增长来自空间曝光还是餐饮能力。
- 最后才回应老板的“加品/加餐/卖茶叶”想法，给出定位和运营建议。

## Calculation Logic

- 去年流水：13.5万元/月，约4500元/日。
- 日毛利：4500元/日 x 70% = 3150元/日；月毛利约9.45万元。
- 租金：46万元/年 ÷ 12 = 约3.83万元/月；按30天约1277元/日。
- 人工：约3.9万元/月；按30天约1300元/日。
- 水电气：4500元/月；按30天约150元/日。
- 已问到的固定费用合计：租金1277 + 人工1300 + 水电150 = 约2727元/日，约8.18万元/月。
- 仅按已问到费用的保本流水：2727 ÷ 70% = 约3895元/日，约11.7万元/月。这个口径未包含平台费、营销费、维修、折旧、耗材、税费等其他费用。
- 去年账面经营剩余：日毛利3150 - 已问到日固定费用2727 = 约423元/日，约1.27万元/月，属于“略赚但很薄”。
- 最近若按5000元/日且毛利率仍约70%：日毛利约3500元；扣已问到固定费用后约773元/日，约2.3万元/月，仍然难覆盖300万元投资的快速回本。
- 投资回收判断：即便按去年约1万多元/月的经营剩余计算，300万元回本周期极长；按最近改善后的粗算也仍是多年级别，且还未扣除其他费用和折旧，所以勇哥判断“回不了多少本/回本遥遥无期”。

## Diagnosis

- 这不是单纯亏损门店，而是“经营现金流略正、投资模型失衡”的案例。门店靠上海郊区较好的环境、包间和接待属性能卖出一定流水，但300万元前期投入和46万元年租金使回本非常慢。
- 经营核心不是餐饮，而是空间。顾客主要买安静、有格调、可谈事/社交的包间环境；茶和简餐是承载空间消费的产品，不一定能变成茶叶零售或正餐利润中心。
- 当前利润薄的主要矛盾是曝光和客单价不足，而不是菜品不够复杂。美团有做，抖音弱，小红书空白；对这种高环境感茶空间，小红书和高审美探店更适配。
- 简餐可以满足附带需求，但厨房和人员能力不支持高端餐饮升级。若把餐饮做高端，需要大厨、更好食材、更高损耗和更高人力成本；菜品不匹配环境或出餐慢，还会带来差评风险。
- 茶叶销售不是优先解法。来店消费的人买的是空间和接待场景，不一定是茶叶爱好者，强行转茶叶零售会偏离顾客真实需求。
- 老板是副业经营，且自用接待价值存在，因此建议不是关店止损，而是在不扩大复杂度的前提下提高曝光、评分、客单价和空间利用。

## Recommendation Pattern

- 先明确定位：继续把它当作“高环境感茶空间/包间接待空间”经营，不要把主线改成餐厅或茶叶店。
- 流量动作：补小红书，找符合调性的本地探店达人和更有品位的内容博主做曝光；美团继续维护评分、评价、套餐和转化；抖音可做但不要只押抖音。
- 客单动作：围绕茶空间增加下午茶、甜点、点心、轻食和适合包间社交的产品，提高客单价，而不是大幅增加后厨复杂度。
- 餐饮动作：保留家常简餐/简单炒菜作为附带消费。可以做清晰的小套餐，但不建议上高端正餐线，除非重新核算厨师、食材、损耗、出餐速度、差评风险和新增客源。
- 包间动作：围绕3小时一场、包间社交、掼蛋/谈事/接待场景设计套餐，重点提高包间使用率、续场率和客单价。
- 茶叶动作：不要把卖茶叶作为核心增长点。只有在不影响空间定位、且顾客主动有需求时顺带成交。
- 经营目标：因为大投资已经发生，后续评价重点应放在“是否持续略赚、是否提升现金流、是否符合老板接待需求”，不要用餐饮快回本模型要求它。

## Reusable Heuristics

- 大投入门店要分两笔账：日常经营是否赚钱、前期投资能否回本。现金流略正不等于项目投资正确。
- 勇哥常用拆账顺序：日流水 -> 毛利率 -> 日毛利 -> 日租金 -> 日人工 -> 日水电 -> 日剩余 -> 投资回本周期。
- 对茶馆、棋牌室、会所型店，先问“卖什么空间、几间房、单场多长、客单多少、翻台/续场怎样”，不要只按餐饮菜品逻辑诊断。
- 顾客买空间时，产品应服务场景。茶、点心、简餐是空间消费的配套，不一定能自然延伸到茶叶零售或高端正餐。
- 增加餐饮品类前必须判断厨房能力、厨师等级、食材损耗、出餐速度和评价风险；高端环境配低能力后厨，反而更容易被挑刺。
- 副业型门店的建议可以是“保持定位、控复杂度、做曝光和客单”，不必一律建议关店；前提是老板清楚回本慢，并认可接待和自用价值。
- 环境型高客单项目的推广渠道要匹配审美和决策方式：评分评价、达人探店、小红书内容，比单纯低价促销更重要。
