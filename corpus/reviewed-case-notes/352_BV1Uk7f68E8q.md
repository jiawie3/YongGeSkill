## Yongge Question Order

1. 先确认品类和症状：是不是自选快餐，是否生意不好，营业额是否上不去。
2. 直接问日流水：一天卖多少钱，上个月卖多少，现在平均每天多少；当用户回答含糊时，追到“平均约1300元/日”。
3. 追毛利率：毛利率多少。用户说不知道后，立刻确认她是不是老板、是不是第一次开店，因为老板不知道毛利本身就是经营风险。
4. 补基础资料：开了多久、多大面积、在哪个城市、月租多少。
5. 拆人员与人工：几个人在店上，5个人工资多少钱，有几个厨师，剩下人员做什么，自己是否在店里。
6. 拆水电：水电多少钱，按开业到月底的水电支出换算日水电，并把水电纳入即时亏损测算。
7. 当场算亏损：用1300元日流水、约50%毛利、6000元月租、3万元月人工、约200元/日水电，推到每天亏约750元、每月亏约2.2万元。
8. 追背景和决策人：谁合伙，以前做什么，用户为什么从医院上班转来帮忙，关键转让和装修过程是谁参与。
9. 追总投资和钱花到哪里：总共投多少，转让费多少，除了19.8万元转让费，剩下40万元花在哪；逐项问加宽费、保证金、喝茶费、装修费、设备费。
10. 用转费倍数质疑接盘合理性：19.8万元除以6000元月租等于33倍，继续问位置是否很好、为什么愿意给这么高转让费。
11. 追用户真正诉求：用户想问能不能改善位置偏、提升营业额，勇哥没有马上给营销建议，而是先追“你有没有参与建店/转让/装修”。
12. 核对家庭决策链：谁去谈转让，丈夫和弟弟是否在场，用户是否知道费用，为什么反对但仍然开。这里勇哥也通过是否打牌/赌博等问题，排查资金决策是否存在异常；ASR 不能确认，不作为事实结论。
13. 现场看位置：要求走到店铺门口、360度转一圈，确认是否一楼、美食城主入口、工厂口方向、工人从哪里进来、工厂距离、午晚餐人流差异。
14. 看竞争结构：问这一圈多少餐饮店、美食城内多少店、前面每一家卖什么、卖多少钱、是否同类、是否有空铺或转让铺。
15. 对比竞品和自家：问最后一家小碗菜能卖多少，追自家开业最高多少、持续几天、为什么后面不行，是不是产品不行。
16. 让用户先说原因：用户提到称重店口碑、会员退款、菜品结构等问题后，勇哥再总结自己的诊断，避免只听用户单一解释。
17. 最后给动作：先查转让费和合同，再停止称重模式，改成和周边低价套餐相近的明码套餐；半个月到一个月测试，做不起来就赶紧转让止损。

## Calculation Logic

- 日流水口径：
  - 当前平均约1300元/日。
  - 折月流水约3.9万元/月。
  - 刚开业5000多元/日只持续两三天，不能作为稳定模型。
- 毛利口径：
  - 用户不知道真实毛利率。
  - 勇哥按约50%毛利做粗算：1300元/日 x 50% = 650元/日毛利。
  - 因为顾客偏夹高价值菜、价格又从2.68元/两降到1.98元/两，真实毛利可能低于这个演示口径。
- 固定与半固定支出：
  - 月租6000元，约200元/日。
  - 人工约3万元/月，约1000元/日。
  - 水电按勇哥现场换算约200元/日，折约6000元/月。
  - 不含老板本人机会成本、装修折旧、设备折旧、转让费摊销、物料损耗和其他杂费时，日支出已约1400元。
- 亏损测算：
  - 650元日毛利 - 200元日租金 - 1000元日人工 - 200元日水电 = -750元/日。
  - -750元/日 x 30天 = 约-2.25万元/月，和勇哥口头“一个月亏两万二”一致。
  - 这个亏损是保守口径；若真实毛利低于50%，或计入装修、设备、老板人工、转让费摊销，亏损更大。
- 基础保本流水：
  - 若按50%毛利，仅覆盖租金、人工、水电约1400元/日，需要日流水约2800元。
  - 当前1300元/日只达到基础保本流水的约46%。
  - 若毛利只有45%，基础保本流水约3110元/日；若要覆盖老板工资、折旧和转费摊销，还需要更高。
- 人效口径：
  - 5个人做1300元/日，约260元/人/日流水。
  - 月人工3万元对比月流水3.9万元，人工约占流水77%，在人效上已经失衡。
- 转让费口径：
  - 转让费19.8万元 / 月租6000元 = 33倍月租。
  - 勇哥口径里，转让费顶多约13-15倍月租；本案约为这个上限的2.2-2.5倍。
  - 若把19.8万元按24个月摊销，约8250元/月、275元/日；按12个月摊销，约1.65万元/月、550元/日。当前650元/日演示毛利几乎无力覆盖转费摊销。
- 总投入口径：
  - 总投入60多万元，扣除19.8万元转让费后，仍有约40万元落在加宽费、保证金、喝茶费、装修、设备和改造。
  - 对一个月租6000元、当前日流水1300元的工厂快餐店来说，总投入与现金流能力严重不匹配。
- 价格调整验证：
  - 从2.68元/两降到1.98元/两后，用户反馈人流略增但营业额没升。
  - 勇哥据此判断，单纯降称重单价无法解决问题：一方面客人仍觉得称重不可控，另一方面低价会继续压毛利。

## Diagnosis

- 这是“高转让费接盘 + 高装修设备投入 + 模型选错 + 新手经营”的叠加问题，不是单纯门头、活动或菜品小修小补能解决。
- 最大硬伤之一是转让费。月租只有6000元，却支付19.8万元转让费，约33倍月租。这个倍数需要非常强的位置、人流和可验证盈利才能支撑，但用户自己也说位置偏，当前流水又只有1300元左右。
- 第二个硬伤是装修和设备投入异常。原店本来就是小碗菜/快餐形态，仍然重装、换新、做改造，导致总投入冲到60多万元。勇哥认为装修已经成为沉没成本，继续追悔没有意义，但要防止再追加错误投入。
- 经营模型与客群不匹配。比亚迪工厂周边工人消费看重便宜、吃饱、价格确定，周边一荤两素/两荤一素等低价套餐已经把心智占住。称重快餐容易让工人觉得贵、不可控，不如10-13元套餐直观。
- 称重模式的毛利也不稳。店家希望顾客夹配菜来提高毛利，但顾客会自然选择好吃、值钱或肉类多的菜；降价后客流略增不代表利润改善，反而可能因为菜品结构让毛利更低。
- 竞争不是“没人流”，而是“人流被更合适的低价模型截走”。现场看位置后，勇哥发现周边餐饮密集、午餐有工厂刚需、同场还有能卖3000-4000元甚至更高的店，所以不能简单归因于美食城没人。
- 位置仍有劣势：店在美食城内较靠后的位置，外面街边和入口处已有大量餐饮店拦截客流，晚上人更少，主要机会集中在午餐工人流量。
- 团队经验不足。老板本人不知道毛利率，合伙人和家属没有餐饮经验，关键决策人没有在现场接受复盘，说明开店前没有完成基本测算和风控。
- 家庭/合伙决策失控放大了亏损。用户说自己反对，但转让、装修和付款已经由弟弟/丈夫推进；勇哥持续追问参与人、转账记录和合同，是在排查“钱是否真实这样花、有没有被中间环节坑”的风险。
- 当前门店不是没有任何机会，而是机会窗口很窄：只有迅速改成当地已验证的低价套餐模型、把菜品和厨师调整好，并在15-30天内看到流水改善，才值得继续试；否则持续亏损会把剩余现金拖光。

## Recommendation Pattern

- 先查账，不先做营销。把19.8万元转让费的合同、转账记录、上一任转让方/房东信息、实际收款路径拿出来核对，确认到底是否真实支付、合同如何约定、有没有异常费用。
- 装修设备已投入的部分按沉没成本处理。不要再围绕“已经花了60万”倒推必须继续做，也不要为了证明原选择正确再追加大额改造。
- 经营动作上，停止继续做称重快餐。勇哥的判断是：这个地方不适合继续做称重，只能改成和旁边类似的一荤一素、两荤一素、一荤两素等明码低价套餐。
- 改品时要把菜品重新调整好。核心不是把称重单价再往下降，而是用工厂工人能快速理解、能吃饱、价格确定的套餐结构去竞争。
- 把招牌和信息递到外门口/入口，降低顾客发现成本。这个店位置在美食城内靠后，不能只等自然进店。
- 前期可以做小活动，比如送小饮料等，用于重新拉回第一批尝试客，但活动只能配合改品，不能替代模型调整。
- 厨师和出品不行就调整，必要时更换。勇哥把菜品、选品、厨师都列为问题，因为当前不是单一价格问题。
- 试错周期要短：半个月到一个月。改成低价套餐、菜品调整、入口引流后，如果流水仍做不起来，就赶紧转让。
- 转让止损要果断。不要因为总投入60多万元就继续每月亏2万多元硬扛；如果改型无效，应尽快看还能转多少钱，接受亏损退出。
- 对后续类似咨询，勇哥不会只给“提升营业额方法”，而会先判断：现有模型是否能覆盖日固定成本、转让费是否过高、商圈主流价格带是否容纳当前品类。若三者都不成立，建议优先止损或极短期改型测试。

## Reusable Heuristics

- 老板说“营业额上不去”时，先问日流水，再问毛利率；如果老板不知道毛利率，要把这本身作为新手风险记录下来。
- 快速亏损公式：日流水 x 毛利率 - 月租/30 - 月人工/30 - 日水电 = 日盈亏。先用保守毛利粗算，再提醒真实毛利未知会让风险更大。
- 基础保本流水公式：日固定支出 / 毛利率 = 日保本流水。本案1400元日支出、50%毛利，对应约2800元日流水；当前1300元明显不够。
- 人效异常要单独看。5个人只做1300元/日，约260元/人/日，说明不是简单“多拉一点客”就能健康，人员结构和模型都要重排。
- 转让费要和月租比，不只看绝对数。勇哥经验里转让费顶多约13-15倍月租；本案33倍属于严重高位接盘，必须追问为什么值这个价、是否有真实盈利支撑、付款是否真实。
- 月租便宜不代表店安全。230平大面积会带来更多人工、水电、设备、备货和管理成本；低租金可能被高人工和低流水完全吞掉。
- 总投入要拆开问：转让费、押金/保证金、喝茶费、加建/扩宽、装修、设备、物料、备用金。拆不清的钱，往往就是踩坑或内部决策风险。
- 原店就是相近品类时，重装要非常谨慎。若小碗菜/快餐底子还在，却重新投入几十万元装修设备，要问这些钱是否真的创造了新增营业额。
- 工厂商圈优先看“便宜、吃饱、确定价格”。面对工人刚需，一荤两素/两荤一素等固定套餐常常比称重更容易成交。
- 称重快餐在低价工厂商圈有双重压力：顾客觉得贵且不可控，店家又容易被顾客夹高成本菜拉低毛利。单纯降每两单价可能增加人流，却不一定增加毛利。
- 现场看位置不要只问“偏不偏”。要看入口、动线、工厂口距离、午晚餐峰值、同场竞品、街边截流、空铺和转让铺。
- 竞品流水要分可靠度。用户口述的“三五千”“一万多”只能作为线索，真正诊断要结合店型、价格、客群、人员和可观察人流。
- 开业两三天的高流水不能作为模型。新店开业活动、尝鲜和亲友消费会制造短峰，稳定期日均才是算账基础。
- 当用户说“我不同意但店还是开了”，要追关键付款人和决策人。家庭合伙店里，真正做决定的人不在场，建议很容易落不了地。
- 异常高转费或装修费出现时，要看合同和转账记录。勇哥不是直接认定被骗，而是要求用户自己核对原房东/转让方、付款路径和合同条款。
- 改型要有截止时间。对已经亏损的新店，可以给15-30天做模式调整；没有明显起色就转让，避免把小亏拖成现金流耗尽。
- 沉没成本不能决定继续经营。60多万元已经花出去，不代表还要继续做错误模型；下一步只看改型后的现金流是否有机会转正。
