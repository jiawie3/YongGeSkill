## Yongge Question Order

- 先问“为什么做糖水、为什么选这里”，直接把品类选择和选址动机拉出来，发现是“喜欢吃”和“离家近”。
- 问基础身份：开什么店、店名、城市、开多久、开之前是否咨询过。
- 问当前经营结果：一天卖多少钱、有没有外卖，追到明确的约300元/日。
- 问毛利率。店主不会算时，勇哥用“卖10元、食材成本3或4元”的例子现场校准，再得到约60%毛利率。
- 问面积、房租、几个人在店、人工按当地工资怎么算、水电怎么算，形成日成本表。
- 立刻把日毛利、日租金、日人工、日水电合并，算出每天亏损和月亏损，让店主看到真实经营口径。
- 问自营还是加盟、总投入、有没有负债、为什么开糖水店、以前是否开过店，判断风险承受和经验水平。
- 进店看菜单和价格，追问技术从哪里学、花了多少钱、定价参考什么。
- 通过视频要求360度看外部环境，追问商圈类型、小区位置、是否市区、方圆一公里小区数、消费是否集中、是否有外来客。
- 继续问周边有什么店、旁边早餐店卖得怎样、是否有空铺、商业体建成多久，用周边店状态校验点位。
- 问店主现在想法和预期目标，确认她只是想把营业额提高到保本。
- 最后再问300元里朋友捧场占比、最近糖水店距离，判断当前流水真实性和品类竞争情况，然后给方案。

## Calculation Logic

- 当前流水：约300元/日。
- 毛利率：约60%，所以日毛利约300 x 60% = 180元。
- 房租：约4500元/月，折合约150元/日。
- 人工：店主和母亲守店，即使自己不拿工资也要计入机会成本；勇哥按很低口径估约200元/日。
- 水电：店主没算，先估约50元/日。
- 当前日盈亏：180元毛利 - 150元房租 - 200元人工 - 50元水电 = -220元/日。
- 当前月亏损：-220元/日 x 30天，约-6600元/月。ASR 有“一月亏660”的漏字迹象，但开头和公式都对应约6600元/月。
- 保本流水：若只覆盖150 + 200 + 50 = 400元/日的租金、人工、水电，且毛利率维持60%，保本流水约400 / 60% = 667元/日。
- 建议目标：从300元/日提升到700、800、1000元/日。700元左右接近保本线，800-1000元才有缓冲，尤其后续上团购、外卖后平台成本会吃掉一部分毛利。

## Diagnosis

- 这不是典型“高租金压死”的店。租金约4500元/月、面积43平，勇哥判断房租不贵，盈亏平衡点不高。
- 主要问题是新手开店前没有完成基本调研：不知道毛利、没算人工水电、不清楚周边店流水、不知道方圆一公里客群，也没有验证糖水在该社区的真实需求。
- 选址逻辑偏个人便利：因为住得近、房子在旁边而选这里；该点位更像社区配套，不是强目的性商圈，外来客少。
- 品类偏窄：全靠糖水，在社区场景里消费频次和客单价容易受限；周边没有糖水店既可能是机会，也可能代表品类需求未被验证。
- 当前渠道缺失：未上外卖，团购刚开始，短视频和本地推广没做，导致自然客流只能支撑约300元/日。
- 但该店仍有试修空间：投入虽约10万元但无明确负债压力，租金低，保本流水约667元/日，不是需要立刻止损的高压模型。

## Recommendation Pattern

- 先补基础平台：把团购尽快完整上线，必要时开推广通，并设计秒杀/引流品，把附近社区用户先拉到店。
- 自己拍短视频做本地推广。勇哥的逻辑是：如果选择糖水这种轻松、少油烟、低后厨负担的品类，就必须用内容和平台获客来弥补自然客流不足。
- 店内加小吃，提高客单价和消费场景，不能只卖糖水。建议方向包括关东煮、炸土豆、酸辣粉或其他适合当地的小吃。
- 对营业执照和品类限制不要只停在“管得严”。需要主动沟通规则、弄清楚能加什么、怎么合规调整，做到灵活。
- 把外卖平台上起来。这个社区点位单靠堂食自然客流不够，要用外卖扩大覆盖半径。
- 短期经营目标不是盲目扩张，而是把300元/日提升到700-1000元/日，先验证保本和轻微盈利能力。
- 如果平台、短视频、小吃和外卖都执行后仍长期低于保本流水，再进入转让或止损判断。

## Reusable Heuristics

- “我喜欢吃”和“离家近”不是开店依据，只能作为兴趣，不能替代品类需求、客群密度和周边竞品调研。
- 新手不会算毛利时，先用单品售价和食材成本举例，把毛利率算清，再把日流水换成日毛利。
- 家人守店也要算人工。即使短期不发工资，也要按当地最低可替代工资计入经营模型。
- 保本流水公式可直接用：日固定成本 / 毛利率。再根据平台扣点、活动折扣和波动加安全边际。
- 社区店的判断重点是“附近人是否够、是否高频、是否愿意专程来”。没有外来客的点位，必须依赖团购、外卖、短视频和社区复购。
- 周边没有同品类店不能直接解释为蓝海，可能是需求不足；要结合最近竞品距离、周边消费集中度、同街店铺生意和空铺情况判断。
- 窄品类甜品/糖水店在非强商圈要考虑加小吃或轻餐，提高客单价和消费理由，但要先核实证照许可边界。
- 对低租金、低保本线、无明显债务的新店，勇哥倾向先给执行清单做修复；对高租金、高亏损、无修复空间的店才更快进入止损建议。
