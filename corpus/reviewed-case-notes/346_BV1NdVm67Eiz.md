- 产品：咖啡、花茶、气泡水、手冲、美式、拿铁，以及薯条、烤翅、沙拉等小吃；最低客单大约20元上下。

## Yongge Question Order

- 先确认店是不是她的、店名和加盟品牌关系，再问现在一天卖多少钱，把流水统一成日/月口径。
- 接着问毛利率、房租、城市，用“1000元流水对应650元毛利”快速建立基本模型。
- 追问几个人在店、老板是否在店、店长和店员分别是谁，马上判断1000元日流水配3个人是否过重。
- 继续问接盘时间、接盘前是否认识勇哥、转让费多少、以前是否在这家店上班，判断是否盲目接盘。
- 细问两名员工工资、店长职责、水电费用，并把老板自己的人工也折进总人工。
- 回头追问面积、怎么找到这家店、前老板报价、接盘时看到的营业数据，以及5月4万元流水是否受节假日影响。
- 要老板360度转场看周边，确认是不是外卖、商圈类型、公园入口、夜市距离、人流强弱和邻近业态。
- 进店看环境、外摆、吧台、厨房、菜单和客单，再听老板自己的产品/引流设想。
- 最后才给诊断：转让费被坑已成事实，短期先砍掉多余店长，长期靠产品结构、客单价和小红书流量修正模型。

## Calculation Logic

- 当前常态流水：1000元/天 x 30天 = 30000元/月。
- 当前毛利：按65%毛利率，约650元/天，约19500元/月。
- 当前成本口径：房租7000元/月 + 两名员工约12000元/月 + 老板自算人工约4500元/月 + 水电约3000元/月 = 约26500元/月。
- 当前盈亏：19500 - 26500 = -7000元/月，约-233元/天。若不计老板人工，现金口径仍大约亏2500元/月。
- 5月流水复算：40000元/月 x 65% = 26000元毛利；对比约26500元成本，约亏500元/月，折合约亏16元/天，所以5月4万元也只是接近保本。
- 店长影响：店长约7000元/月。若砍掉店长，在5月4万元流水口径下可从接近亏平转为约6500-7000元/月利润；勇哥据此说一年约8.4万元、约两年回17万元转让费。
- 保本流水：保留当前3人模型时，保本月流水约26500 / 65% = 4.08万元，即约1360元/天；砍掉店长后，保本月流水约3万元上下，当前1000元/天才有机会接近不亏。
- 转让估值：勇哥认为这个点位17万元转让费严重偏高，真正转让的是客流、位置和经营结果，不是前老板说的装修金额；设备和二手咖啡机的残值不足以支撑高转让费。

## Diagnosis

- 短期亏损核心是人效错误：1000元日流水不需要老板、店长、店员三个人，店长7000元/月把本来接近保本的小店压成亏损。
- 接盘错误在于把“环境好”和“前老板给看的4万月流水”当成价值，没有扣除节假日、人工、水电、老板自己的人工和转让费回本周期。
- 点位不是完全没机会。公园、夜市、外摆、晚上散步人群给了空间消费场景，但现有客流和客单支撑不了17万元转让费。
- 店型本质是卖空间、卖环境、卖拍照和停留体验。只卖20元左右咖啡，客人坐很久，客单太低，座位效率不足。
- 客群有偏差风险。中年客打牌会把店带成茶馆逻辑，和“漂亮饭/下午茶/小红书打卡”的年轻女性客群不是同一套模型。
- 加盟价值不清晰。ASR 显示这是加盟店，但现场没有体现品牌自带流量；勇哥借此提醒，若加盟只给流程、合同和普通物料，没有品牌、配方、流量和运营支持，加盟费就是高风险支出。

## Recommendation Pattern

- 第一动作：先取消店长岗位，老板自己在店里盯运营，保留一个员工即可；1000元日流水下，采购和物料管理不值得单独花7000元/月。
- 不建议立刻关店。若日流水能稳定做到1300元左右，并砍掉店长，模型有机会转正；17万元已是沉没成本，先用经营动作止血。
- 产品结构要从单杯咖啡升级为空间型消费：白天咖啡、下午茶、绵绵冰/刨冰可以做；晚上要增加融合菜、简餐、小食等能提高客单的产品。
- 目标客单要从20元左右拉到40-50元。因为客人不是买完即走，而是在店里停留，必须用套餐、餐食和拍照型产品提高单桌消费。
- 夜市放二维码海报不是主策略。它只能辅助提醒，不能替代产品力和内容种草。
- 流量主战场放小红书，不优先放抖音日常视频。原因是目标客群以女性打卡、下午茶、漂亮饭为主，小红书图文和探店内容更匹配。
- 小红书内容要先把产品和环境拍好，用图文、真实体验文案、本地博主打卡来拉目标客群；不是简单发日常流水视频。
- 如果后续生意起来，再调整吧台/咖啡机位置等动线问题；前期先处理人效、产品和流量。

## Reusable Heuristics

- 接盘店不要按装修报价估值。转让费买的是可验证客流、位置、证照条件、设备残值和稳定利润，不是前老板说自己花了多少钱装修。
- 看转让店流水必须区分节假日月和常态月。5月有五一，4万元流水不能直接外推全年。
- 餐饮算盈亏要把老板自己的人工计入，否则会把“自己白干”误判成盈利。
- 保本公式：保本月流水 = (房租 + 人工 + 水电 + 老板人工 + 其他固定成本) / 毛利率。先算当前模型，再算砍人后的模型。
- 1000元日流水的小店一般不配店长，除非店长能直接带来高于工资的新增流水、管理半径或老板脱身价值。
- 空间型咖啡/下午茶店不能只看毛利率，还要看停留时间、座位效率和客单价；客人坐很久时，20元客单很难支撑房租人工。
- 公园、夜市、河边、外摆等环境资源只有转化成拍照内容、目标客群和高客单产品才有价值。
- 小红书适合女性打卡、漂亮饭、下午茶、空间型门店；抖音日常视频如果没有强产品钩子和转化路径，通常不是优先战场。
- 高转让费预算常常不如投到产品打磨和本地种草。勇哥举例的逻辑是：少花钱接盘，把钱用于产品升级和小红书博主推广，可能比买一个“环境好”的壳更有效。
