## User Facts Extracted

- 咨询者想做“啤酒交易所”，核心是自助果味啤酒/精酿类饮品设备，后台按销量做价格涨跌，包装成类似“炒股买酒”的娱乐体验。
- 项目不是明确加盟，而是刷抖音和去青岛台东步行街看到类似模式后照搬。咨询者说没有交学习费，主要是找厂家生产设备、找啤酒/物料供应方。
- 选址在广西柳州某步行街/市中心商圈，咨询者说人流大、旅游客和本地人大概五五开，旁边约四五十米有美食街。勇哥现场视频判断：人流确实有，但不等于店内消费。
- 计划点位原有小集装箱/摊位，后续要拆掉上他的设备。设备形态为直径约4米圆柱体，占地 ASR 对应约13平米。
- 租赁成本最终被勇哥追问清楚为：租金约3万元/月；转让费10万元；押三付三，押金约9万元、预付租金约9万元。前面 ASR 出现“40一年/40万吗/3333”等混乱，应以后续澄清口径为准。
- 设备厂家报价40万元，设备生产好后运到现场组装，咨询者说不用额外装修。设备内部涉及冷库、扎啤/酒桶、更换酒桶等，ASR 细节粗糙。
- 首批酒水/物料金额未谈清，咨询者与厂家/供应方谈的是约一周结一次账，因此勇哥没有把首批物料压货计入核心投入。
- 人员计划为2人，每人约4000元/月，合计8000元/月。虽然说“自助”，但酒桶一桶约40杯，卖完要有人换酒，不能完全无人化。
- 电费按厂家说法约150元/日，水费口径不明确或很少。毛利率咨询者报65%-70%，勇哥后续按65%保守计算。
- 项目已经不是纯咨询阶段：咨询者已付给摊位/转让方10万元，又付给厂家10万元定金，合计已经付出20万元。设备在更早的另一个场地计划中已经生产好，年前另一个场地毁约后才转到当前场地。
- 咨询者自称给自己留了“尺寸谈不下来可以退”的后手，但勇哥认为钱已经付出、设备已做，退款并不受他单方面控制。
- 咨询者是江苏人，长期在广西跑业务/做销售，妻子孩子不在当地；妻子认为地方远、不同意。咨询者说自己没有固定工资，靠自己做业务提成，一年约三四十万元口径。
- 周边调研不足：他提到看过茶饮店小票，知道某品牌店单量很高，但没有记录小票起始号，也没有系统蹲点；对冰糖葫芦/茶饮/蜜雪类门店的真实日流水、员工数、人效缺乏验证。
- 现场周边有空铺，周日约下午三点街上有人流，但勇哥让他观察实际消费：走路逛街的人多，正在店里买、喝、吃的人并不对应增加。

## Yongge Question Order

- 先问项目属性：这是自创、加盟，还是跟别人学；再问“啤酒交易所”到底是什么意思，价格涨跌由谁决定，顾客为什么要为这种机制买单。
- 追问咨询目的和阶段：是想评估位置、产品能不能做，还是已经交钱了；确认他已经付了摊位钱和厂家钱后，勇哥把问题从“能不能做”转为“还能不能止损”。
- 追问城市和点位：在哪个城市、具体放在步行街哪里、是否拆原集装箱、设备形态和占地面积。
- 追问租赁成本：年租/月租怎么付、押几付几、有没有入场费/中介费/转让费。勇哥在这里纠正咨询者把租金和转费混在一起说的口径，拆成租金3万元/月、转让费10万元、押金9万元。
- 追问模式来源：去哪里学、是否花钱、是青岛台东步行街看到的模式，还是抖音刷到后觉得“打通了任督二脉”。
- 追问投入结构：设备多少钱、装修多少钱、厂家怎么交付、首批物料怎么结算，再合计出这个项目看起来至少约68万元的前期资金压力。
- 追问运营成本：几个人、人工多少钱、水电多少、毛利率多少，然后马上算日固定成本和保本流水。
- 追问风险控制是否真实：转让方和厂家各付了多少钱、付了多久、设备是否已生产、所谓尺寸谈不下来能退有没有约束力。
- 让咨询者现场360度看街区：判断是不是步行街主街、周末和平日客流差异、是否街中心、周边门店有没有真实消费。
- 追问周边店真实数据：茶饮/小吃/知名品牌一天卖多少、数据从哪里来、是不是看小票、有没有看起始号、有没有蹲点。
- 用同街品牌店做对比：问蜜雪类大品牌在同样街区有几个员工、一个员工的产能和人效大概多少，以此反推这个商圈的消费容量。
- 最后做回本和极端亏损账：先按咨询者乐观预估算三年回本，再加入冬季淡季、新奇感衰退、雨季和500元/日低流水情景，逼出“不能接受”的结论。

## Calculation Logic

- 前期总投入口径：设备40万元 + 转让费10万元 + 押三付三约18万元 = 约68万元。该口径不含未确认的首批酒水物料，也不含后续可能追加改造。
- 已付沉没成本口径：转让/摊位定金10万元 + 厂家定金10万元 = 已付20万元。勇哥认为这已经不是“问问看”的阶段，而是已经进入止损谈判阶段。
- 月固定成本：租金30000元/月 + 人工8000元/月 + 电费150元/日约4500元/月 = 约42500元/月。
- 日固定成本：按30天折算，租金约1000元/日，人工约267元/日，电费约150元/日，合计约1416-1417元/日。
- 保本流水：按65%毛利率，1416 / 0.65 = 约2179元/日。也就是每天至少卖到约2200元才覆盖租金、人工、电费，还没有回设备和转让费。
- 杯数口径：咨询者/厂家预估每天约200杯，每杯约15元，即3000元/日流水。按65%毛利，日毛利1950元；扣日固定成本约1416元，日净约534元。
- 月利润口径：534元/日 x 30天约16020元/月。勇哥用约1.6万元/月作为乐观经营利润。
- 乐观回本：勇哥后续用约59万元投资口径做回本账，59万元 / 1.6万元/月 = 约36个月，即3年左右。这里59万元与68万元总投入之间存在押金/已付租金等口径差异，ASR 未完全清晰；但勇哥意图是先按偏乐观账算，仍然很慢。
- 淡季调整：勇哥按一年约4个月冬季/冷天淡季，每月亏3万元；一年少12万元，三年少36万元。36万元再除以1.6万元/月，约再多22个月。总回本约36 + 22 = 58个月，约4.8年。
- 低流水情景：如果热度过去后一天只卖500元，毛利为500 x 65% = 325元/日；325 - 1416 = 日亏约1091元；30天月亏约3.2万-3.3万元。
- 勇哥没有进一步计入雨季、设备损耗、啤酒损耗、推广费用、达人费用、新奇感衰退后的长期客流下降、设备报废残值低等风险，因此上述账已经是偏保守的风险下限。

## Diagnosis

- 核心问题是“网红模式照搬 + 高固定成本 + 高设备沉没成本”的组合风险。咨询者不是先验证模型再投钱，而是被抖音和外地案例刺激后先交钱、再找人评估。
- “啤酒交易所”的涨跌价机制不等于真实需求。勇哥判断顾客买饮料时更关心值不值，不会长期为了价格涨跌的噱头买单；新奇感即使有效，也可能只有一两个月。
- 人流被误当作购买力。步行街有人逛、美食街在旁边，并不代表会买自助啤酒；勇哥现场让他看周日旺时有多少人在实际消费，强调“路过”不是“转化”。
- 场景不可直接复制。青岛台东、啤酒文化、旅游氛围、啤酒节城市记忆，与柳州本地步行街并不等价；外地成功案例不能只复制设备和外壳。
- 周边调研不合格。咨询者知道一些茶饮品牌单量，却没有蹲点、没有看完整小票号、没有分平日周末、没有记录员工数与时段；数据不足以支撑68万元投入。
- 固定成本明显偏高。3万元/月租金叠加2个人工和150元/日电费，要求每天约2200元流水才能保本；对一个新奇型、冷饮/啤酒型、季节性明显的项目压力很大。
- 设备资产风险很高。40万元定制设备一旦生意失败，二次转让价值和残值很低；勇哥提醒供应商可能在设备定金或设备报价里已经赚走主要利润。
- 所谓“预留后手”不可靠。定金已经付、设备已经生产，退款需要合同和对方配合，不是咨询者设一个尺寸条件就能保证退出。
- 家庭和主业风险被放大。咨询者妻子不同意、家庭不在当地，而他自己的主业一年还能有三四十万元收入；用一个重资产网红项目去赌改行，会威胁家庭现金流。

## Recommendation Pattern

- 直接建议及时止损，不要继续进场安装，不要再补设备尾款、租金或改造款。
- 先把能退的钱尽量退回来：转让费/摊位定金10万元能拿回多少拿多少，厂家定金也应按合同谈退或谈停；如果对方拒退，应回到合同、付款凭证和证据链处理。
- 不要把“已经付了20万”当作必须继续投的理由。勇哥的逻辑是：20万损失已经很痛，但继续把68万投满，后面可能变成每月亏3万和设备报废。
- 回归主业，不要把网红项目当作主业困难时的救命稻草。主业即便不理想，也比不懂模型、远离家庭、重资产押注一个未验证项目更可控。
- 如果以后再看类似项目，必须先蹲点：平日/周末分开记录客流、进店率、成交单数、小票号起止、客单价、员工数、人效、天气和季节差异。
- 对外地网红案例要拆场景，而不是看外壳：当地有没有同类消费文化、喝酒氛围、夜经济、旅游动机、年轻客群停留时间、周边消费链路。
- 对自助设备类项目，要先问设备残值和替代方案：如果不做了能不能卖、卖给谁、多少钱、拆走运输多少钱；残值差的设备不能按资产看，应按沉没成本看。
- 所有定金和转让费都应在模型核算、合同退出条款、场地审批、电力接入、天气淡季测算之后再付；先付款后验证，所谓“后手”通常失效。

## Reusable Heuristics

- “人流大”不是餐饮结论，只是输入条件；必须继续问进店率、成交率、客单价和复购，否则人流只能证明有人路过。
- 外地成功案例要先拆解不可复制条件：城市文化、旅游属性、消费习惯、季节气候、夜生活和商圈结构。复制设备不等于复制需求。
- 网红/新奇型项目的账，不能按全年稳定旺季算。至少要单独测算热度期、平稳期、冬季/雨季、消费者看腻后的低流水情景。
- 高毛利不代表好项目。先用固定成本除以毛利率算保本流水，再问这个流水需要多少杯、多少人、多少转化率。
- 定制设备报价越高，越要问失败后的残值。不能二次利用、不能轻易转卖的设备，本质上接近一次性赌注。
- 定金付出之前才叫风控；钱付出、设备生产之后再说“条件谈不下来就退”，多数只是心理安慰。
- 周边大品牌是很好的反向标尺：看同街成熟品牌的员工数和人效，可以估计本商圈真实消费容量，避免高估自己的新项目。
- 若乐观账都要3年回本，加入淡季、热度衰退、设备残值和家庭现金流后，通常不应继续投。
