## User Facts Extracted

- 咨询对象：延安市烧烤老板，约40岁，已经有一家老店，正在看二店位置。
- 老店情况：老店开了四年多到五年，面积约500-600平，ASR 多处显示约500多平；主做烧烤，是当地有名的烧烤店。
- 老店流水：淡季目前约70-80万元/月，夏天可到约100万元/月；工作日约2万多元/日，周五周六可能3万多元/日。视频开头和中段反复确认老店年流水已做到约1000万元。
- 老店利润：老板口述淡季一个月大概能挣十几万元，追问后确认约15万元/月。老店租金、完整费用表未在 ASR 中确认。
- 老店能力：老板本人是烤肉师傅/产品把关人，负责产品、即时发现问题、研发和标准；店内约40多人，已把烤肉、烤菜、出餐、点单等岗位拆分，并有一定流程化和标准化基础。
- 新店位置：延安市新城区的社区底商，属于城区内，不是城边；有市政停车位；对面和周边有大社区，后段确认周围类似社区约五六个。老店距此约8公里。
- 新店面积：拟租一楼约720平，其中约100平公摊或布局不佳区域；多个铺面可打通。
- 新店租金：报价约55元/平/月，720平约3.96万元/月，约47.5万元/年；老板口述大概50万元/年；年付、无押金。
- 新店投入：装修加设备预计约200万元；勇哥把首年租金等算进去后认为至少约247万-250万元，再加营销费用可能到270万-280万元。
- 新店人员：老板预计新店员工规模和老店差不多，约40-50人；按45人、每人约5900-6000元/月，理论人工约27万元/月，但老板说现在实际工资约20万元/月。
- 新店水电气：老板估约2万-3万元/月，勇哥按3万元/月进入算账。
- 毛利：烧烤毛利率老板口述可做到约60%-62%。
- 老板动机：想把老店作为体验店，再做一个更大的形象店/旗舰店，提高品牌知名度，围绕陕北/延安特色烧烤做主题和装修，之后再发展小店模型、连锁化。
- 团队动机：老板认为核心团队也需要成长、平台和验证；团队成员想小入股，老板倾向先不让他们投钱，未来赚钱后给干股或股份，让大家一起挣钱。
- 当前位置评价：勇哥后段明确说，作为社区商圈和本地知名烧烤品牌，这个位置本身“没毛病/OK”；反对点不在选址本身，而在扩张节奏和重资产投入。

## Yongge Question Order

- 先把镜头稳定下来，要求老板站定，“我问你回答”，避免一边走一边看铺导致信息散乱。
- 先确认咨询阶段：是已经开店还是找新位置；确认这是为二店找位置。
- 看铺面边界：确认准备租哪几个门面、一楼范围、是否可打通。
- 问城市，再问面积、楼层、公摊和布局，先判断物业基础。
- 问租金，把“55”追成单位：是55万元/年还是55元/平/月；随后用720平乘55元/平/月换算月租和年租，并追问付款方式、押金。
- 问为什么拿这么大的店，转入扩张动机，而不是只停留在位置好坏。
- 问老店面积、开店年限、老板年龄、老店与新店的定位关系，确认这是从成熟单店往二店/旗舰店扩张。
- 问装修设备投入，随后把200万元装修设备、首年租金和营销费合并成总投资口径。
- 问为什么开二店，追出“形象店、旗舰店、提升知名度、未来连锁”的战略目标。
- 问股东结构，再问员工规模、人工、水电气、毛利率，算出新店保本流水。
- 问现有老店是不是生意很好，再追日流水、月流水、淡旺季、月利润，判断老店真实成熟度。
- 问老店股东、家人态度，轻触决策约束，但没有把家庭意见作为主诊断。
- 现场转圈看周边：问社区底商、城区/城边、停车、为什么选这里、旧商户情况、距老店多远。
- 问老板本人在老店做什么，确认他是否仍是产品和标准的关键人。
- 问老店岗位是否拆分、是否已经标准化流程化，区分“单店能力强”和“连锁组织能力强”。
- 听完长期规划后，再集中输出判断：不是先否定位置，而是先讲扩张会带来的管理成本、精力分散、连锁风险和资金消耗。
- 老板提出团队成长和市场占位后，勇哥再追核心团队是否入股、是否真成熟，并回到老店坪效和排队强度，判断还没到必须扩张的阶段。

## Calculation Logic

- 新店租金：720平 x 55元/平/月 = 39,600元/月，约3.96万元/月；年化约47.52万元，和老板口述“约50万元/年”一致。
- 新店租金日口径：约3.96万元/月 ÷ 30 = 约1320元/日。
- 新店人工口径一：45人 x 6000元/月 = 27万元/月。勇哥先用这个数提醒人员规模很重。
- 新店人工口径二：老板修正说现在工资约20万元/月，勇哥后续保本测算显然按约20万元/月人工进入。
- 新店水电气：按3万元/月，约1000元/日。
- 新店固定成本粗算：租金3.96万元/月 + 人工20万元/月 + 水电气3万元/月 = 约26.96万元/月，约8987元/日。这里未包含平台费、营销费、物耗、维修、税费、管理人员奖金、折旧和财务成本。
- 新店保本流水：若毛利率按60%，约8987元/日 ÷ 60% = 约14,978元/日，和勇哥口述“至少要做到14977一天才能保本”一致。
- 若按45人 x 6000元的理论人工27万元/月，新店已问固定成本会变为约33.96万元/月；按60%毛利率，保本流水约1.89万元/日。这不是勇哥当场主算口径，但提示人工一旦按满员发生成本，安全边际会更低。
- 新店首年现金占用：装修设备约200万元 + 年租约50万元 = 约250万元；若加开业营销，约270万-280万元。
- 老店流水：淡季70万-80万元/月，约2.3万-2.7万元/日；夏季约100万元/月，约3.3万元/日；年流水约1000万元。
- 老店淡季利润：约15万元/月，约5000元/日。因老店租金、营销、折旧等没有完整披露，这个利润只能作为老板口述经营结果，不宜反推完整模型。
- 对比逻辑：老店一个月淡季约15万元利润，而新店开业要一次性占用约250万-280万元；新店还要老板复制一个40-50人的组织，保本线至少约1.5万元/日。即使位置不差，现金、管理和精力风险都远高于单纯租金账。

## Diagnosis

- 这不是亏损门店，也不是“生意不好求救”的案例，而是一个强单店老板在高峰期想重资产扩张。老店年流水约1000万元、淡季仍能挣约15万元/月，说明单店基本盘不错。
- 反对二店的核心原因不是点位差。勇哥认可该位置作为社区商圈、配合本地知名烧烤品牌可以成立；但他判断现在开720平、250万以上投入、40-50人团队的旗舰店，扩张节奏过急。
- 老板把“单店产品和标准化能力”误认为“连锁组织能力”。老店能做起来，和老板本人把控产品、研发、标准、岗位拆分有关；一旦开二店，老板精力被分散，管理成本和管理难度会明显上升。
- 老板的战略目标带有形象店、品牌升级、连锁化和团队平台诉求，但这些目标不一定需要立即用一个720平重资产门店来验证。现有老店还没有达到长期满负荷、严重外溢、不开分流店就承接不了的程度。
- 勇哥后段用老店坪效和排队强度校准老板的自信：500-600平烧烤店淡季70万-80万元/月，生意很好但不是极端爆满；节假日最高排队五六十桌，也不是持续排队到必须扩店。
- 团队想成长、想赚钱可以理解，但不能把“给团队平台”直接等同于“马上开大店”。如果模型不够成熟，让员工入股或跟着扩张，反而会把团队利益绑到高风险项目里。
- 连锁化风险被低估。勇哥强调做一个好单店和做8个、10个连锁店是两种基因；烧烤又涉及食材新鲜、出品稳定、人员流动和高温辛苦岗位，一旦扩张后某个环节出问题，品牌和成本都会被放大。
- 当前更好的增量来自老店优化。老店已经有40人团队，继续提高工作日流水、周末峰值、顾客体验、员工稳定和内容传播，新增流水大部分会转化为利润，比另起一个250万级项目更稳。

## Recommendation Pattern

- 暂缓二店，不建议现在签这个720平大铺。不是因为位置绝对不能做，而是现阶段没有必要用250万-280万元和一个新40-50人团队去赌扩张。
- 再等两年左右，继续打磨老店。把产品、服务、岗位、出品稳定性、员工激励、顾客体验和内容能力做到更扎实，再决定是否复制。
- 不急着做加盟和连锁。老板目前更像强单店经营者，而不是已经具备连锁加盟基因的组织；先把单店模型做到极致，比急着开8家、10家更重要。
- 把老店作为品牌阵地继续放大：继续做好短视频和直播，老板本人出来站台，讲陕北/延安特色烧烤，把单店名气做长红。
- 先提升老店经营上限：围绕工作日流水、周末流水、翻台、排队承接、服务体验和顾客复购做优化。勇哥的意思是，原店基础上每多出来的流水，往往比新店流水更容易变成利润。
- 团队激励不必靠新店解决。可以通过老店利润提升、奖金、干股或阶段性分红让核心团队受益；在模型没有充分成熟前，不建议让团队成员投钱入股承担二店风险。
- 如果未来再开二店，前置条件应更硬：持续外溢的客流、老店脱离老板本人仍稳定运行、核心管理层能独立带店、现金储备更厚、单店标准可复制、供应链和食安管理能承受多店。
- 对这个具体铺位的判断应分层：社区底商、停车和周边社区可以支持烧烤消费；但位置可做不等于现在该做，选址判断必须服从战略节奏和组织能力。

## Reusable Heuristics

- 好单店不等于可连锁。单店成功常常来自老板本人、位置、口碑和现场控制；连锁成功需要组织、供应链、培训、督导、食安和财务管控。
- 勇哥做扩张审查时，先问新店物业和投入，再回头验证老店成熟度：老店面积、开业年限、日/月流水、利润、老板角色、岗位标准化、是否有持续客流外溢。
- 扩张不是只算保本流水，还要算首年现金占用、管理复杂度、老板精力分散和品牌事故放大效应。
- 对大面积餐饮店，要把租金、人工、水电气都拆成日口径，再用毛利率倒推保本日流水；本案约1.5万元/日只是基础保本，未包含大量隐性费用。
- 当老板说要做“形象店、旗舰店、连锁模型”时，要追问真实目的：获客外溢、品牌面子、团队激励、融资加盟、市场占位，还是老板焦虑。不同目的对应不同解法。
- 位置“没毛病”不代表应当签约。咨询判断要区分点位可行性、项目必要性和当前时机。
- 老店尚未长期爆满、尚未严重排队外溢、尚未脱离老板稳定运行时，优先提升原店坪效和利润，不急着用二店承接想象中的增长。
- 团队成长可以通过老店利润提升和内部激励解决；不要为了给团队平台而把团队推入未经充分验证的重资产扩张。
- 餐饮老板在生意顺时最容易高估复制能力。越是赚钱时，越要慢下来查模型、查组织、查食安、查现金储备。
- 对烧烤这类高人工、高现场、高食安敏感品类，扩张后的小问题会被放大；食材新鲜、出品稳定和人员流动，是比装修形象更核心的连锁风险。
