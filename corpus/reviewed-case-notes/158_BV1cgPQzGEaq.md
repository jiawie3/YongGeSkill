- 产品设想是魔术主题清吧：两人会变魔术，现场给客人表演，也计划拍抖音、小红书，下午直播唱歌或变魔术，周五周六视情况请驻场。附近竞品酒馆有驻场，用户观察到流水大约从两三千到四千多不等，但样本和 ASR 都较粗。

## Yongge Question Order

- 先确认品类和场景：开什么酒吧、是否清吧、魔术主题具体怎么呈现。
- 再问城市、商圈、面积、展示面和租金，判断基础租金压力和点位形态。
- 追问付款方式、转让费、中介费、哪些设备留下、哪些要买，先把进入成本拆清楚。
- 对“上家说赚钱”提出反问：为什么三个人退出、是不是利润分配问题、对方说法是否可能有水分，避免把转让方叙述当证据。
- 继续盘装修、灯光、灯箱、吧台、音响、首批酒水物料，把总投资汇总到17-18万元量级。
- 问几个人干、当地人工多少钱、合伙比例怎么分，发现合伙人打算直接按出资比例分红后，先纠正“工资和分红要分开”。
- 问水电网、毛利率，并用租金、人工、水电和65%毛利率快速算出保本流水。
- 问有没有开过清吧、现在是否在酒吧学习、为什么想开酒吧，用来判断创业动机和操盘能力。
- 让咨询者展示魔术，观察“主题卖点”是否真的能形成体验和内容素材。
- 现场看街区：是不是主街、离主街多远、晚上和周末人流、周边烤肉/酒吧是否有客、竞品是否有驻场和大概流水。
- 最后问经营计划：怎么做营销、抖音/小红书谁拍、谁出镜、是否直播、是否请驻场、朋友能投入多少时间。

## Calculation Logic

- 房租口径：7500元/月，年租金90000元；押一付一年意味着首期现金约90000 + 7500 = 97500元。
- 前期投入口径：房租押金约97500元 + 轻装修疑似约20000元 + 吧台/音响/杂项设备约30000-40000元 + 首批酒水物料约10000元 + 零碎二手物件约2000元，勇哥归纳为约16.95万元，实际按17-18万元看。
- 月固定成本口径：租金7500元 + 3人工资12000元 + 水电网6000元 = 25500元/月。这里的人工把两位合伙人的基础劳动价值也按当地工资先计入。
- 保本流水：计划毛利率65%，则月保本营业额约25500 / 0.65 = 39231元/月；按30天约1308元/天。勇哥现场简化为约1300元/天达到平衡点。
- 合伙分配逻辑：先把参与经营者的工资作为经营成本计完，再按出资比例分红。不能因为一方出资比例高，就把双方每天劳动的工资价值混在利润分配里，否则实际参与者容易吃亏。

## Diagnosis

- 这不是亏损门店复盘，而是开店前评估。核心判断不是“酒吧一定赚钱”，而是这笔17-18万元投入在无负债前提下是否值得试。
- 点位有一定匹配度：台东商圈、周边餐饮和酒吧氛围、晚上和周末有人流，适合做小型氛围酒吧。但该铺不在主街，不能只靠自然客流，必须靠内容和运营补足。
- 最大风险不在单项租金，而在获客能力和执行能力：魔术主题能不能被目标客群喜欢，短视频和小红书能不能推出去，现场表演能不能转化为到店理由。
- 咨询者有比普通新手更好的准备：已经去酒吧学习调酒、运营和表演，自己能全职参与，卖点不是纯想象。但仍然没有独立开店经验。
- 合伙结构需要提前规范。2:1出资或分红并不能替代工资、岗位责任、决策权和退出机制，否则容易重演上家“利润分配没谈清”的问题。
- 上家“挣钱”的说法只能作为线索，不是证据。真正的验证要来自多时段蹲点、竞品流水观察、预热内容反馈和试营业数据。

## Recommendation Pattern

- 勇哥给的是有条件的“可以尝试”：前提是两位合伙人没有负债，17-18万元亏得起，并且咨询者愿意全职操盘。
- 轻投入做，不要为了主题感盲目加重装修和设备。能用的留下，不能形成体验或出片价值的设备不要买。
- 开业前就开始做内容，不要等店开好再找客。可围绕“两个魔术师开酒吧”、选址、装修、调酒、魔术表演、试营业故事持续拍抖音和小红书。
- 把短视频、小红书、直播当成核心经营动作，而不是附属宣传。这个店的成败重点在能否把魔术内容做成流量和到店理由。
- 继续在现有酒吧环境里验证自己：能不能适应夜场节奏，魔术表演在真实客人面前反应如何，哪些桥段适合转化为消费场景。
- 合伙协议开业前先定清楚：工资标准、出资比例、分红顺序、岗位分工、拍摄和出镜责任、决策机制、退出和亏损承担。
- 每天用约1300元流水作为最基础的保本线观察，试营业后按日记录客流、客单、毛利、人工和活动效果，再决定是否增加驻场等额外费用。

## Reusable Heuristics

- 开店前评估先问硬事实：品类、城市、商圈、面积、租金、付款方式、转让费、设备归属、装修和首批物料，而不是先听创业故事。
- 上家说赚钱不能直接采信。要反问为什么退出、利益关系是什么、是否存在误导，再用现场客流和竞品数据交叉验证。
- 合伙店必须区分三种钱：工资是劳动回报，分红是投资回报，能力股或资源股要单独约定。工资先计入成本，剩余利润再按股权分。
- 保本流水的快算公式：月固定成本 / 毛利率 = 月保本营业额；再除以30天得到日保本流水。
- 年轻、无负债、无家庭刚性压力、总投入在可承受范围内，且创始人已深入一线学习时，可以把小店当作可控试错；借钱、高杠杆或完全外行则不适用。
- 主题酒吧本质上是内容和体验生意。位置给基础人流，真正的增量来自可传播内容、现场互动和稳定运营。
- 对“技能型卖点”要现场验证。会变魔术、会唱歌、会调酒不等于能开店，关键是能否持续生成客人愿意拍、愿意分享、愿意复购的体验。
- 支路或非主街点位要按夜间和周末验证，不只看商圈名气；周边业态、排队情况、竞品驻场和竞品流水都要纳入判断。
