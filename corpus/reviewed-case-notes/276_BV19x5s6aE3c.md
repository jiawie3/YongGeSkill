## Yongge Question Order

- 先让用户把镜头对准目标铺位，不要离开现场；确认城市、市区属性、铺位面积、门头宽度、柱子和实际门口形态。
- 进入租赁条件：年租多少、月租折算、怎么付款、有没有入场费/中介费/打点费用。
- 确认项目品类和品牌：准备做什么、什么酸奶、是否加盟、加盟费和各项费用是多少。
- 拆品牌公司收费：18万元具体包含什么，设备多少钱，加盟费多少，是否包含物料，是否为代理还是单店。
- 拆开店总投资：装修谁做、装修多少钱、首批物料多少钱、35万元到底是总投资还是仅给公司、是否包含房租。勇哥没有纠缠到精确账，而是归类为三四十万元级项目。
- 查人和运营能力：用户年龄、以前做什么、是否做过餐饮、准备几个人做、是否本人守店、是否把店当副业。
- 现场口算经营模型：按4人人工、当地工资、水电、50%毛利率，反推日保本流水，再问用户自己预期日流水。
- 追品牌来源和动机：怎么知道这个品牌、为什么选择它、是网上看到还是熟人推荐、产品差异点是什么、有没有被“当下看起来火”吸引。
- 查加盟公司背书：品牌公司是否看过位置、是否已经收钱、怎么判断这个位置可以。
- 让用户360度转镜头看商场内部：确认楼层、业态、商场面积、开发主体、预计开业时间。
- 查外部商圈：周边百货、购物中心、步行街、酒店、人流强弱、是否老商圈退化。
- 查招商进度和租赁逻辑：有无免租期、招商是否饱和、为什么这个铺还没出租、是否因为用户关系提前留铺。
- 让用户看负一楼和外部环境，但重点不陷在一楼/负一楼二选一，而是追问整个商场的运营逻辑和客流来源。
- 追问商场核心优势：除了古风打卡、换装、表演，还引入了哪些能带客流的头部品牌、超头部品牌、各品类头部品牌。
- 根据用户回答做判断：如果临近开业大牌仍只是“在谈”，招商逻辑不成立；如果商场大多是吃喝玩乐，很可能开成美食城而不是强购物中心。
- 最后给到两个层面的建议：第一，当前商场和位置未知数太多，可以先不做；第二，如果非要在开荒商场做饮品，应尽量拿头部品牌，而不是小众网红酸奶。

## Calculation Logic

- 年租16万元，月租约13333元，日租约444元。
- 押二付三意味着启动时仅租押现金就约66667元；这不是每月损益成本，但会增加开店现金压力。
- 勇哥按4个人核算人工：4000元/月/人 * 4人 = 16000元/月。用户原本估计商场店可能要6个人，勇哥先用更保守的4人模型。
- 水电按4000元/月估算。
- 按4人模型，月固定成本约33333元：租金13333 + 人工16000 + 水电4000。
- 日固定成本约1111元：33333 / 30。
- 毛利率按50%，则保本月流水约66666元：33333 / 50%；保本日流水约2222元：66666 / 30。这个数与勇哥现场口算一致。
- 如果按用户原先估计的6个人，人工变为24000元/月，月固定成本约41333元，50%毛利下保本日流水约2756元。
- 用户预期日流水6000-7000元，折合月流水约18万-21万元。按50%毛利，月毛利约9万-10.5万元，扣4人模型固定成本后，理论经营利润约5.7万-7.2万元/月。但勇哥没有把这个当结论，因为这个流水完全建立在新商场客流和小品牌热度能兑现的假设上。
- 前期投入口径较粗：给公司约18万元 + 装修约3-5万元 + 首批物料约3-5万元 + 租押约6.7万元 + 其他入场/打点约1万元，整体就是三四十万元现金风险。即使保本线只有2222元/天，也不代表值得投，因为还要看稳定流水、投资回收期和失败概率。

## Diagnosis

- 这不是一个已经亏损门店的止损案例，而是未开业商场开荒前的投资拦截案例。核心风险不是“酸奶这个品类毛利够不够”，而是位置、商场、品牌、经营者能力四个未知数叠加。
- 单店模型表面上能算得过去：若真能稳定做到6000-7000元/天，账面利润很好。但问题在于用户没有证明这个商场开业后有足够客流，也没有证明该小众酸奶品牌能持续带来转化。
- 新商场开荒首先看招商质量。勇哥反复强调商场需要头部/超头部品牌和各业态强品牌带客流；如果临近开业仍是大量“在谈”，就不能把这些品牌当成未来客流来源。
- 文旅、古风、换装、歌舞表演、网红打卡只能说明主题，不等于稳定消费。若没有强零售、强餐饮、强娱乐等业态组合，容易变成一次性打卡或普通美食城，无法支撑三四十万元加盟店的回本确定性。
- 外部老商圈已经退化，周边百货和购物中心人流不旺。新商场若没有特别强的招商和运营能力，不能天然继承“市区老商圈”的客流。
- 小众网红酸奶品牌风险高：短期热度、产品新鲜感、公司宣传、线上视频都不能替代真实门店数据。酸奶、糯米、水果等产品差异容易被跟进，用户不能只看“现在有点火”。
- 加盟公司看位置并说可以，不构成可靠背书。加盟公司有收加盟费、设备费和服务费的利益，位置判断必须由投资人自己用商场招商、客流和竞品数据验证。
- 用户把开店当副业，只准备守两个月；而新商场开荒和饮品店都需要强运营、强盯店和快速调整。这种投入方式和项目难度不匹配。
- 用户有工程背景和商场关系，能拿到铺位，但这不等于具备商业判断、品牌选择和门店运营能力。勇哥的底层判断是：来访者是看到别人或网上项目赚钱而心动，并没有完成必要调研。
- 如果明明有商场关系、能参与开荒，却把机会用在一个小众网红品牌上，风险结构很差。开荒本身已高风险，品牌就应尽量选择更稳定、更能自带客流的头部品牌；如果拿不到头部品牌，也要把这当作能力和资源不足的信号。

## Recommendation Pattern

- 直接建议：当前不要交加盟费，不要急着签约进场；先回到自己的工程主业，把这个项目当高风险投资看待。
- 如果还想做，第一步不是继续研究酸奶口味，而是向商场要硬资料：已签约品牌清单、各楼层业态图、主力店/头部品牌合同状态、开业排期、招商完成率、免租/租金政策、预计客流依据。
- 对“在谈”的品牌一律按未发生处理。只有已签约、确定开业、确定楼层和面积的品牌，才能纳入客流判断。
- 至少等商场开业后观察真实客流和消费转化，再决定是否进场。若一定要提前锁铺，也要用免租期、退出条款、低成本试营业或短租快闪降低风险。
- 品牌选择上，抛开具体位置，如果必须在饮品/咖啡/茶饮里选，优先选择头部或更成熟品牌。勇哥举的方向包括瑞幸、霸王茶姬、一点点、茉莉奶白等；不是说这些一定能做，而是原则上比小众网红品牌更适合承担开荒风险。
- 不要被“早期品牌”“原始股东”“刚开始有红利”说法打动。加盟商不是股权投资人，承担的是门店亏损和加盟沉没成本，不能用投资叙事替代单店生意验证。
- 经营模型必须设底线：保本约2222元/天只是最低门槛，实际要覆盖三四十万元投入回收、淡旺季波动、开业热度衰减、平台费用、损耗和管理成本，目标应显著高于保本线且能稳定。
- 如果拿不到头部品牌、拿不到商场真实招商数据、本人也不能长期守店，这三个条件同时存在时，应停止项目。

## Reusable Heuristics

- 未开业商场开店，先问招商，不先问铺位。没有确定主力店和头部品牌，铺位再好也只是图纸上的好位置。
- “在谈某大牌”不能计入判断；只有签约、落位、确定开业的品牌才算商场势能。
- 文旅/网红/古风/打卡是主题，不是生意模型。要继续追问谁来、为什么来、来几次、来后消费什么。
- 开荒商场和小众品牌不要叠加风险。若位置是新商场，就尽量用头部品牌降品牌风险；若品牌是小众网红，就应放在成熟高流量商圈验证。
- 加盟公司说位置可以，只能当销售意见，不能当投资依据。位置判断要来自租金、人流、竞品、业态、动线、商场招商和自己的保本模型。
- 保本线不是投资理由。保本2222元/天只说明不亏损的最低流水，不能说明能回本，也不能说明开业后能稳定做到。
- 用户预估“能卖6000-7000元/天”时，必须追问依据：同商场已开竞品、成熟商圈同类店、品牌真实后台、周边人流、转化率、客单价和复购频次。
- 把店当副业、只守两个月，却做新商场开荒饮品店，是典型运营错配。越是不确定的位置，越需要老板深度在场。
- 做工程、有关系、能拿铺，不等于能做餐饮。跨行用户要先补商业认知：品牌筛选、商场逻辑、单店模型、真实数据、风险退出。
- 头部品牌是否愿意授权，也是项目质量筛选器。越容易随便放加盟的品牌，越要警惕它是不是靠找加盟商赚钱，而不是靠门店模型赚钱。
- “早期红利”“原始股东感”“小众新品类”常见于加盟销售话术。餐饮咨询中要把叙事拉回单店现金流、稳定客流和可退出性。
