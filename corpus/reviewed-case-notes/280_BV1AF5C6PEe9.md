# Case 280: 县城广式糖水店小赚但缺推广

## User Facts Extracted

- 店在水富/水附县城，ASR 地名粗糙；老板原来做自创奶茶品牌，在县城做了约 8 年。
- 当前店 2022 年开，原本是奶茶店，因为周边奶茶品牌店增多、竞争变大，刚改成广式糖水店约 1 个月；现在糖水和奶茶都卖。
- 老板自述这个位置租金较贵，但也是县城里比较好的地段之一；她担心挣不到钱，想判断是否继续做、是否优化，还是改方向。
- 上个月流水 4.1 万元，约日均 1367 元。勇哥追问后理解为目前不是新手开店，也不是完全没流水。
- 毛利率约 60%。按月流水 4.1 万元计算，月毛利约 2.46 万元。
- 房租 7000 元/月，实用面积约 50 平方米。
- 店里 3 个员工，不含老板本人。老板另有一家奶茶店，偶尔过来帮忙。
- 3 个人工约 1 万元/月，水电气约 1500 元/月。
- 按勇哥口径，当前月粗利润约 6000 元：2.46 万毛利 - 7000 房租 - 1 万人工 - 1500 水电气 = 约 6100 元。
- 老板没有仔细算过账，且刚转糖水店才 1 个月。之前做奶茶已经越来越不赚钱，甚至偏亏，所以才转型。
- 老板计划上绵绵冰，设备投入 ASR 中约 1 万元；勇哥建议该加可以加，但买不起新的就找二手，降低新增投入。
- 位置周边有商场、古茗/蜜雪/霸王等饮品品牌，附近有小学，接送孩子时段有人流。附近吃的有麻辣烫、面馆、包子店等，但勇哥没有看到特别强的刚需餐饮密度。
- 竞品古茗等老板估计堂食加外卖日流水约 4000-5000 元，到店约 70-80 单；自己店约 30-40 单，客单价 20 多元。该组数据与月流水存在一定口径差异，需视为粗略描述。
- 县城目前基本没有同类糖水店。曾有一家位置偏的糖水店开了约半年后关掉；老板认为自己是当前更好位置上的第一家。
- 老板自己拍过抖音，但推广很少。勇哥看后判断视频质量一般，产品和店铺介绍很少，更多是在讲个人情绪和故事。

## Yongge Question Order

- 先确认品类：是不是卖奶茶，现在为什么从奶茶改成糖水，是否糖水和奶茶都卖。
- 接着确认城市层级和市场环境：是否在县城，奶茶竞争为什么变大。
- 追问经营结果：刚改糖水多久，上个月流水多少，日均大概多少。
- 让老板说清楚来咨询的具体问题：是租金贵、位置是否值得继续、还是想看优化方向。
- 回溯经营经验：以前是什么品牌，开了多久，是否自创，老板是否餐饮新手。
- 进入算账：毛利率多少，房租多少，面积多大，几个人在店，老板本人是否算在人工里，是否还有别的店。
- 继续补成本：3 个人人工多少钱，水电气多少钱，再据此判断当前是不是已经能小赚。
- 对比转型前后：之前奶茶是否赚钱，为什么干不过周边品牌，判断改糖水不是随意换品类，而是对原奶茶竞争的应对。
- 明确决策点：老板是想继续优化，还是纠结是否改行；同时询问新增绵绵冰设备投入。
- 要老板 360 度转店外环境，现场看街区、商场、品牌店、学校、餐饮店和人流结构。
- 追问竞品表现：周边饮品品牌一天大概卖多少，到店多少单，老板自己多少单、客单价多少。
- 检查推广动作：有没有做推广，抖音内容拍了什么，县城有没有其他糖水店，门口能否做外摆。
- 在下结论前再次确认数据是否准确，然后给出“位置没问题、能挣钱”的判断。

## Calculation Logic

- 现状流水：4.1 万元/月，约 1367 元/日。
- 毛利：4.1 万元 x 60% = 2.46 万元/月。
- 已知固定成本：房租 7000 + 人工约 10000 + 水电气约 1500 = 18500 元/月。
- 勇哥粗算利润：24600 - 18500 = 6100 元/月，口语中归为“一个月能挣 6000 块”。
- 粗保本流水：18500 / 60% = 约 3.08 万元/月，约 1028 元/日。该保本点未纳入平台抽佣、税费、折旧、损耗、推广费和老板工资，实际安全线应更高。
- 目标不按大店高流水要求，而按县城单店可承接的现实目标看：日流水 2000-3000 元，即月流水约 6 万-9 万元。
- 若维持 60% 毛利，日流水 2000-3000 元对应月毛利约 3.6 万-5.4 万元；扣已知 1.85 万固定成本后，粗利润约 1.75 万-3.55 万元/月，再扣其他未确认费用。
- 订单倒推只能粗看：老板称客单 20 多元、当前约 30-40 单/日，和 4.1 万月流水并不完全吻合，可能是堂食/外卖口径、时段、ASR 或估算偏差造成，不能据此精确建模。

## Diagnosis

- 这不是典型“位置烂、租金压死、必须止损”的案例。按已知口径，店刚转糖水 1 个月已在粗算层面小赚，且位置在县城中心街区，周边有商场、饮品品牌和学校，人流基础不差。
- 老板的核心问题是信心不足和账没算清。她感觉租金贵、挣不到钱，但勇哥先用流水、毛利、租金、人工、水电把结果算出来，发现店还没有到必须改行的状态。
- 奶茶转糖水的逻辑成立：原奶茶面对古茗、蜜雪、霸王等品牌竞争，老板的自创品牌优势下降；糖水在当地暂无强同类竞争，反而形成“县城第一家/稀缺品类”的测试空间。
- 当前短板不是产品投入不够，而是顾客不知道这家店卖什么、糖水有什么价值、产品长什么样、价格和位置在哪里。老板的视频没有把产品、套餐、店铺和消费理由讲清楚。
- 糖水不是所有人都需要的刚需餐，但县城只支撑一家店时，不需要教育全城所有人，只要抓住学生、年轻人、爱尝鲜人群，就有机会做到日流水 2000-3000 元。
- 新增绵绵冰可以作为夏季产品补充，但必须控制投入。需求还在验证期，优先用二手设备或低成本方式补产品线，避免因为信心不足而盲目重投入或盲目改行。

## Recommendation Pattern

- 先继续做，不要马上改行或放弃。理由是位置可用、同类竞争弱、粗算已经小赚，且转型才 1 个月，还没有完成充分推广测试。
- 把“推广不清楚”当成第一优先级，而不是继续纠结租金。推广目标是让县城消费者知道：这里有正宗广式糖水、有哪些产品、味道和价格如何、店在哪里、为什么值得来。
- 达人和老板自有 IP 同步推。先找县城本地达人来拍，再扩展到市区、周边城市里做得好的达人；花几千元集中拍产品、套餐、门店和消费场景，勇哥判断这笔钱值得。
- 自有 IP 视频用三段式：第一段用“水富奶茶姐”等本地身份做开场；第二段讲店铺、产品、味道、价格和位置；第三段给到店理由、折扣或召唤语。
- 不要只拍老板个人故事和情绪。个人 IP 可以保留，但必须服务于产品成交：每条内容都要让顾客看懂卖什么、为什么好吃、多少钱、去哪买。
- 绵绵冰可以上，但设备别重投。买不起新的就找二手，先补足产品吸引力和夏季消费场景，再根据推广后的客流反馈决定是否继续加码。
- 短期目标定在日流水 2000-3000 元，不要用 5000、8000、10000 元的高流水压自己。先验证县城是否能支撑这一家糖水店。

## Reusable Heuristics

- 老板说“租金贵、挣不到钱”时，先算账再判断。用月流水 x 毛利率 - 房租 - 人工 - 水电，先区分真实亏损、微利焦虑和账目不清。
- 刚转型 1 个月的店，不能只凭老板信心下滑就建议止损。若位置、毛利、成本和早期流水尚可，应先做一轮有强度的推广验证。
- 县城新品类/第一家店的核心风险通常是认知不足，不一定是没有需求。诊断时要问：当地有没有同类、是否有年轻客群、是否只需要支撑一家店。
- 自创奶茶遇到强连锁围攻时，换到差异化甜品/糖水可能比继续硬拼奶茶更合理，但必须重新做品类教育和产品展示。
- 位置判断不能只听“贵”。要现场看周边品牌、商场、学校、人流、刚需餐饮和竞品单量，再结合保本流水判断租金是否真的不可承受。
- 低信心老板常把问题归因于品类或位置，勇哥会先拆成三层：账面是否活着、位置是否能承接、推广是否把产品讲清楚。
- 新设备投入要跟验证阶段匹配。需求未跑通时，能用二手、低成本补品类就不重资产加码；但如果产品线缺口影响成交，也不能完全不补。
- 推本地小店时，达人探店和老板 IP 各有作用：达人负责快速触达和信任背书，老板 IP 负责长期本地记忆点和复购召唤。
