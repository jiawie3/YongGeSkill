## User Facts Extracted

- 咨询人在云南昆明，原本做商业摄影，约 30 岁，非餐饮出身。
- 门店已经租下且正在装修，形态是广场上搭建的集装箱/箱体铺，约 30-35 平。
- 租金 5500 元/月，年付，折合 6.6 万/年；押金 5000 元；无转让费。
- 计划做“漂亮酒”/网红鸡尾酒，强调颜色好看、拍照打卡；还想搭配蛋糕、甜点、薯条等小食。
- 项目灵感来自小红书，看到类似品牌/项目做得好，并称已有两三百家门店。勇哥没有把这个当作有效验证。
- 咨询人自己不会调酒；朋友正在学或可教，且本店涉及与朋友合伙。
- 装修费约 2.5 万；首批物料约 5000 元；设备金额 ASR 粗糙，疑似 2 万左右但未确认；勇哥复述总投入约 12.1 万。
- 计划两个人经营，即咨询人和朋友。勇哥按昆明两个人工约 9000 元/月估算，再给水电杂费粗算约 1500 元/月。
- 毛利率按 50% 估算。
- 位置在综合批发市场/市场采购贸易试点一类区域，市场里有服装、男女装、玩具、鞋帽等，不算昆明市区核心商圈。
- 店在喷泉广场附近，晚上有夜市摊，摊位多为饰品、衣服、鞋子等；白天人很少，晚上自然流量当事人估算一晚几个小时约 200-300 人，且不含摆摊的人。
- 周边可见停车场、公共驿站/公交站、天桥、花店、卖被子、旅游相关、便利店等；广场上餐饮氛围弱，且该区域不允许做油烟。
- 咨询人设想的客户是批发市场里的员工，尤其是“小姐姐”，以及附近几百米到一公里内住的人。他还提到市场里有好几万人，但未证明这些人会经过、停留或购买鸡尾酒。
- 市场与店之间有路/通道，市场有很多门，当前点位只是其中一个门附近。即使市场人多，也要经过“从这个门出来、愿意过来、愿意喝酒、愿意买鸡尾酒”多层筛选。

## Yongge Question Order

- 先问遇到什么问题、准备做什么品类，再追问城市，确认是昆明的网红鸡尾酒/漂亮酒。
- 立刻问铺子是否已经租、让对方站远看店形态，确认面积、月租、付款方式、押金、是否有转让费。
- 问咨询人以前做什么，判断其餐饮经验来源和新手程度。
- 追问产品和能力：是不是鸡尾酒、有多漂亮、自己会不会调酒、是加盟还是学技术、谁教、是否和朋友合伙。
- 追问投入结构：装修多少钱、设备多少钱、首批物料多少钱、总预算是否够。
- 追问经营组织和保本账：几个人做、两个人工多少钱、水电杂费多少、毛利能做到多少，然后现场推保本流水。
- 追问开店动机：为什么想做漂亮酒，在哪里看到别人这么开，是否只是小红书种草。
- 要求 360 度看点位：看招牌、市场类型、广场、喷泉、夜市摊、左右邻居、背后交通、是否有客流。
- 追问夜间客流口径：晚上到底卖什么摊、来逛的人多不多、200-300 人是一分钟还是一晚上，排除摆摊人后判断真实自然流。
- 追问目标客群：到底是批发市场顾客还是员工；如果是员工，为什么他们下班后不回家、不吃饭、不接孩子，而要到店里喝一杯鸡尾酒。
- 追问消费环境：漂亮酒需要拍照氛围，但周边是夜市摊、被子、便利店、旅游店和嘈杂广场，是否能支撑“漂亮”。
- 当用户提出白天做，勇哥再问白天做什么、白天有没有人、是否能改其他项目、油烟限制是否允许烧烤。
- 最后给止损方向：不要继续做鸡尾酒；若想挽回损失，只能考虑无油烟、低投入、顺路型小吃并边做边转。

## Calculation Logic

- 租金：5500 元/月，年付约 `5500 * 12 = 66000 元/年`；另有押金 5000 元。
- 初始投入：装修约 25000 元，首批物料约 5000 元，设备金额 ASR 未确认；勇哥复述总投入约 121000 元。若按年租 66000 + 押金 5000 + 装修 25000 + 物料 5000 倒推，设备/其他约 20000 元，但这里只作为疑似口径。
- 人工：两个人按昆明口径粗算约 9000 元/月。即使两人是老板，也应按机会成本算进模型，否则会低估保本压力。
- 水电杂费：勇哥粗算约 1500 元/月。
- 基础月固定成本：租金 5500 + 人工 9000 + 水电杂费 1500 = 约 16000 元/月。
- 毛利率：按 50% 估算。
- 基础保本月流水：`16000 / 50% = 32000 元/月`。
- 基础保本日流水：`32000 / 30 = 约 1067 元/天`。ASR 中勇哥说法类似“一天一千零六十六才能达到平衡点”。
- 这个保本点只覆盖租金、人工、水电杂费，不含装修设备折旧、前期投入回收、营销、损耗、平台费用、淡旺季和老板真实收入。
- 当点位一晚自然流只有约 200-300 人，且其中大量并非目标饮酒客群时，日保本 1000 元以上并不是看似很低；关键在于有效客流和购买动机不足。

## Diagnosis

- 核心问题不是鸡尾酒配方，而是品类消费场景和点位人群完全不匹配。漂亮酒依赖拍照、社交、氛围和目的性消费，但该点位更像批发市场外的广场夜市，环境嘈杂、杂货摊多、餐酒氛围弱。
- 咨询人把“小红书上好看、别人门店多”当成需求验证。勇哥反复打断这一点，是因为种草内容证明不了本地人会在这个点位、这个时间、这个场景下掏钱。
- 咨询人把“市场里有好几万人、小姐姐多”直接等同于目标客群。勇哥实际在拆漏斗：有多少人从这个门出来，有多少人过马路/走通道，有多少人愿意喝酒，有多少人愿意买一杯鸡尾酒带走。
- 对下班生活轨迹判断错误。批发市场员工下班后的主需求通常是回家、吃饭、接孩子、休息、约会或聚餐，不是继续停留在工作场所附近和同事喝一杯微醺鸡尾酒。
- 白天无客流，晚上只有几个小时有摊市，且自然流估算只有 200-300 人；这个流量不足以支撑一个需要氛围、停留和拍照的网红酒饮模型。
- 周边业态反向证明点位属性：花店、被子、旅游、便利店、饰品衣服鞋子夜市等，都不强化年轻女性酒饮消费心智。勇哥用这些邻居判断目标客群画像偏中老年/刚需/低停留，而不是精致酒饮客。
- 油烟限制使转品空间变窄。若能做烧烤等夜市强需求品类，可能还有另一套模型；但该点位不允许油烟，限制了更适合夜间广场客流的刚需餐饮。
- 已租已装修带来沉没成本压力，但继续按漂亮酒投入只会扩大损失。勇哥判断这是常识和生活经验问题，不是再补一点专业运营就能解决。

## Recommendation Pattern

- 直接否定当前漂亮酒/鸡尾酒方案，不建议继续装修、学技术和追加投入。
- 如果要做私域或靠内容引流，不需要 5500 元/月的广场铺；勇哥建议这种逻辑应该租 1000-1500 元/月的便宜空间，而不是承担高可见铺位成本。
- 咨询人应回到自己的摄影主业，不要把短视频/小红书种草当作创业依据，也不要让朋友继续围绕这个店学习鸡尾酒。
- 若租约已经形成、想尽量挽回损失，只能考虑与过路客更匹配、无油烟、低投入、低客单、顺手购买的小品类，例如钵钵鸡、清补凉、冰粉等。
- 这些替代品类也不是“能赚钱”的强承诺，而是为了边做边转、降低空置损失、尽快把铺子处理出去。
- 后续若仍想做酒饮或网红饮品，必须先验证真实消费场景：目标人群在哪里休闲、是否愿意为拍照和微醺绕路、周边环境是否能拍照、夜间停留时间是否足够、竞品和同类店实际流水如何。

## Reusable Heuristics

- 总人流不是有效客流。要连续拆成“经过本门口的人、愿意停下的人、符合品类的人、愿意付款的人”。
- 选址不能只看人群标签，要看生活轨迹。员工下班后的第一需求往往是离开工作场所，而不是在工作场所旁消费一个非刚需品类。
- 网红品类必须匹配环境。卖“漂亮”和“打卡”的店，周边如果杂乱、嘈杂、低拍照价值，产品再好看也很难成立。
- 小红书种草和品牌门店数量不能替代本地验证。线上案例要落到本地客流、动线、消费动机、客单、复购和时段。
- 做保本账先用 `月固定成本 / 毛利率 / 30` 算日保本，再用真实有效客流和转化率判断，而不是只看“每天卖一千多好像不高”。
- 市场/园区/商场“有好几万人”要拆门、拆路线、拆时段。很多人并不会经过你的门口，更不会为了低频非刚需品类绕路。
- 不允许油烟的铺子，转品空间会明显受限；签租前必须确认哪些品类不能做，不能等项目失败后才发现无法改强需求餐饮。
- 如果核心获客靠私域或内容，不必为自然流位置付高租金；位置租金必须服务于自然客流转化。
- 已租已装修时，建议应围绕止损和退出，不应为了证明自己没错继续追加大投入。
- 勇哥常用反问法：把用户想象的消费行为放回日常生活里问“你下班会不会专门绕过去买一杯”，用常识校验需求是否真实。
