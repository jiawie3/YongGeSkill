## User Facts Extracted

- 咨询者此前做汽车行业约10年，做到职业经理人，约3个月前停止卖车，现想第一次自己创业开店。

- 城市为新疆阿克苏市。点位在商场五楼，海底捞正在装修，预计7月左右开业；咨询者选的位置在海底捞斜对面，口述离海底捞门口约2-3米。

- 合伙人为从小到大的朋友，投资五五分。朋友所在地 ASR 前后不完全清楚，只能确认也在新疆一带，是否同城共同经营不稳。
- 产品方向是果茶、轻乳类饮品，预计单价约15元/杯。咨询者认为自创品牌比加盟毛利更高、成本更低、价格更低。
- 装修加设备预计约15万元；首批物料口述6万元，后面又说多了可以3万-5万元；勇哥汇总这个点位总投入约25万元。

- 自报毛利70%-75%，勇哥要求按70%算，并进一步提醒70%在奶茶里偏高，实际可保守看68%左右。
- 商场客流口述为月均约40万人、日均约1.2万人，周末约1.4万-1.5万人；数据来自商管和当地询问，不是咨询者长期独立计数验证。
- 商场一至三楼已有奶茶店，四楼也有奶茶店，五楼当前没有奶茶店。商场内外合计约6家奶茶/饮品竞争店。
- 竞品 ASR 可辨认的有霸王茶姬、茉莉奶白、冰雪时光、沪上阿姨等，另有若干品牌名识别不稳。咨询者口述霸王茶姬约15000元/天，茉莉奶白约8000-10000元/天；这些数据由咨询者观察或询问得来，未被独立核验。

## Yongge Question Order

- 先问“现在遇到什么情况”，让咨询者说明背景：汽车职业经理人转行、准备自己干、看好一个市场/商场位置。
- 看到海底捞后马上确认：是不是要开海底捞，还是海底捞只是对面参照物；随后让对方直播看位置。
- 迅速锁定基础选址变量：城市、具体楼层/商场、面积、长宽、实际可放设备宽度、后场是否只是站人通道。
- 追问租金结构：月租多少、有没有押金、是否有入场费和中介费，再核对商场实际给的是1.2米还是2.4米，以及哪些区域因防火门不能放设备。
- 问准备做什么品类，听到奶茶后追问为什么不加盟、联系了哪些品牌、为什么这些品牌不能做。
- 听到“自创品牌”后立刻拆来源：怎么自创、谁研发、朋友是否有加盟店、朋友加盟哪个品牌、有没有亏钱、研发团队是谁、要不要发工资、研发费用是否要算。
- 追问合伙可信度：朋友是什么关系、是否见过研发团队、是不是都听朋友说；用这些问题识别“熟人背书”和“未验证资源”。
- 转入开店账：装修设备多少钱、首批物料多少钱、总投资多少、朋友投多少、是否会负债。
- 继续拆固定成本：准备几个人、每人工资多少、水电多少、毛利率多少、单价多少，并要求用保守毛利算保本流水。
- 查老板本人能力边界：问是否职业经理人、有没有分辨力、卖车停多久、年龄多少，由此判断他是否具备从零创业能力。
- 在确认尚未付款后，让对方360度转镜头看商场现场，并追问商场客流、数据来源、楼层奶茶分布、商场内外一共多少奶茶店、竞品每天卖多少、这些数据是数的还是听来的。
- 最后用竞品和楼层反问：凭什么认为自创品牌能干过这些品牌；对方提出五楼餐饮格局变化、价格低、毛利高时，勇哥逐条拆掉。

## Calculation Logic

- 月租：12500元/月，折约417元/天。
- 人工：4人 x 4000元/月 = 16000元/月，折约533元/天。
- 水电：约3000元/月，折约100元/天。
- 已确认的刚性月成本约31500元/月，折约1050元/天，不含老板工资、损耗、物料报废、营销、税费、设备折旧和商场其他杂费。
- 毛利率：咨询者报70%-75%，勇哥先按70%算，又提醒70%在奶茶里偏高，保守可按约68%理解。
- 按70%毛利覆盖31500元/月固定成本，理论最低保本流水约45000元/月，即约1500元/天。
- 按68%毛利覆盖31500元/月固定成本，理论最低保本流水约46300元/月，即约1540元/天。
- 勇哥现场给出的保本口径约1750元/天，应理解为在68%-70%毛利基础上再加入杂费、损耗或保守余量后的粗算。这个口径下月保本流水约52500元。
- 客单价约15元/杯。若按1750元/天保本，需要约117杯/天；若按1500元/天保本，需要约100杯/天。这个杯数只是覆盖账面固定成本，不代表真正赚钱。
- 总投入：装修设备约15万元，首批物料3万-6万元，押金约1万元，加上其他开办费，勇哥汇总约25万元。合伙五五则双方各约12.5万元，但具体出资节奏未确认。
- 与竞品对照：霸王茶姬口述约15000元/天，茉莉奶白约8000-10000元/天；但这些是强品牌在更成熟楼层/位置的表现，不能直接套给五楼自创品牌。自创店的关键不是保本线低，而是能不能被顾客纳入选择。

## Diagnosis

- 这是一个应在付款前止损的案例。风险尚未变成沉没成本，勇哥抓住“钱还没给”这一点，直接把建议前置到不要开。
- 点位物理条件差：15平长条形、宽度受限、后场部分受防火门/通道约束，设备、排队、出杯、备料和顾客感知都会受影响。
- 商场五楼不是天然奶茶位。奶茶购买通常依赖顺手买、强曝光和品牌心智；五楼餐饮层更多是饭点目的性客流，非饭点可能没人。
- 海底捞开业和五楼餐饮品牌进驻不等于奶茶自带流量。顾客到五楼吃饭，不代表会走进一个无品牌心智的新奶茶店。
- 奶茶、咖啡这类品类头部效应很强。消费者说喝奶茶时，脑子里通常先出现已有品牌；自创品牌如果不在顾客的备选答案里，就还没进入“比价格、比毛利、比产品”的阶段。
- 咨询者把“价格更低、毛利更高、成本更低”当优势，但勇哥认为这些优势成立的前提是顾客先进店、先选择你；如果顾客连门都不进，后端毛利没有意义。
- 加盟品牌不能进，反而说明点位不满足成熟品牌的开店条件。咨询者把它转化成“自创品牌机会”，勇哥把它看成风险信号。
- 合伙和研发资源没有被验证。所谓十几人研发团队、朋友能出产品、研发不收费，更多来自朋友口述；创业者没有亲自核实团队、成本、产品验证和责任边界。
- 咨询者有职业经理人经历，但勇哥区分了“管理已跑通的企业/门店”和“从零造一个商业模型”。他可能适合接手成熟体系，不适合在陌生餐饮品类里从零创业。
- 虽然保本流水看似只需1500-1750元/天，但这个账不能单独支持开店。真正难点是自创品牌在五楼如何获得首批顾客、复购和品牌信任。

## Recommendation Pattern

- 直接建议不要开，不付款、不装修、不签约，把风险止在开店前。
- 不要因为海底捞、五楼餐饮格局变化、商场客流数字和几个头部品牌高流水，就推导自创奶茶可以成立。
- 如果想进入餐饮，先去相关品类上班或进入成熟体系学习，补齐品类认知、选址、供应链、出品、门店动线和消费者心智，而不是从汽车职业经理人直接跳到自创奶茶。
- 对合伙朋友提供的研发、品牌、产品和团队资源，要先实地验证：见团队、看样品、算研发费、明确股权和责任、验证已有门店真实流水和利润。不能只凭从小到大的朋友关系下注25万元。
- 未来若再看奶茶/饮品点位，应优先问成熟品牌是否愿意进、为什么不进；如果成熟品牌因面积、楼层、动线不进，自创品牌更要谨慎，而不是认为自己成本低就能补位。
- 继续创业前先做低成本验证：在同城同楼层长期计数竞品订单、分时段人流、饭点与非饭点转化、顾客是否愿意为无名品牌停留；没有这些数据，不进入签约和装修阶段。

## Reusable Heuristics

- 开店前咨询的最大价值是“没给钱就能停”。只要还没付款，建议可以非常硬，不必为了已投入沉没成本找补救方案。
- 奶茶/咖啡先看品牌心智和曝光位，再看毛利。顾客不把你列入选择，70%毛利也只是纸面数字。
- 头部品牌不愿意进的点位，要反向验证原因。面积、楼层、动线、消防通道、门头曝光限制，都是成熟品牌筛掉点位的信号。
- 商场餐饮层客流不能直接等于饮品客流。饭点人多和全天可卖奶茶是两件事，必须分时段看人流和停留意愿。
- 海底捞、连锁餐饮进驻只能说明楼层可能有饭点目的性客流，不能自动给旁边小品牌导流。
- 自创品牌说“更便宜、更高毛利、更低成本”时，要先问获客入口。没有进店率，低价和高毛利都无法兑现。
- 熟人合伙要用更严格的验证，不是更少的验证。朋友承诺的研发团队、产品、资源和不收费，都要落实到人、合同、样品、成本和责任。
- 职业经理人能力不等于创业能力。会管理成熟机器，不代表会从零验证品类、选址、品牌和现金流模型。
- 保本流水要折成日口径、杯数口径和获客口径。12500月租、4个人、3000水电看似不高，但每天100杯以上对五楼无名奶茶并不容易。
- 竞品高流水不能照抄。霸王茶姬、茉莉奶白等强品牌的15000元/天或8000-10000元/天，证明的是品牌和位置的组合，不证明旁边自创店可行。
- “本地没有这个品牌/项目”不是机会本身。对高品牌依赖品类，顾客没听过往往是阻力，不是空白红利。
