- Reliability: 中高。ASR 有方言和品牌/品类误识别，周边店名不完全可靠；关键经营数字较清晰，包括原店日流水约500-800元、原店租金约5000元/月、目标铺租金15万元/年、转让费35万元、4个人、人均工资3500元/月、水电约4000元/月、综合毛利按55%算、目标铺保本流水约1800元/日。周边店流水多为老板口述或听员工说，未蹲点核实。

## User Facts Extracted

- 咨询者约38岁，周口某县城口径，本地人；有妻子和孩子，老大约20岁，老二约12岁，老三约1岁半。家里有人在店里帮忙，后续想让儿子接现有面馆，自己和妻子另做一个店。
- 现有店在县城商业街，已开约4年，主做陕西手工面，去年加过米线，老板说米线还可以。现店约70多平方米，店内有楼梯，厨房空间不大，约9张桌子。
- 现有店日流水老板前面说七八百，后面勇哥复述为五六百；可按约500-800元/日理解。老板认为生意不好，但说不清具体原因。
- 现有店租金约6万元/年，即5000元/月。老板曾考虑不转、找合伙人，或假设转让大概10万到15万，后面勇哥按“自己的店转10万”来反推。
- 目标铺与现店在同一条街，相距约50米。目标铺一楼约150平方米，整栋三层必须全租，二楼三楼实际用不上或用途不清。
- 目标铺租金15万元/年，即12500元/月；一年一付，老板口述无押金；转让费35万元。开头和后文都显示勇哥认为这个铺子存在被快招/加盟类项目割过后的接盘风险，但具体品牌、合同和前任经营数据 ASR 未核清。
- 老板还没有想清楚要做什么品类，提到冒菜，也提到去成都学技术或加盟，但没有确定学习对象、品牌和预算。当地人是否认知冒菜，老板也回答不上来，只说朋友以前在外地做过。
- 商业街层面：老板说这是县城中心商业街之一，县城类似商业街有两个，此街客流算多，两边餐饮较集中。周边有麻辣烫/麻辣香锅、自助小火锅、烤肉、杨国福、面类等店，但具体店名 ASR 粗糙。
- 周边流水为老板口述：麻辣烫/麻辣香锅类约3000元/日；自助小火锅约3000元/日，依据是员工来吃面时说的，不是蹲点；部分品牌店约2000多元/日，非品牌店约1000多元/日。勇哥反复追问数据来源，提示这些数字未被验证。
- 街上同时存在多家转让/空铺，一家铺子空了一年半左右。后面老板提到有的转让费可能是两千、三千、几千元口径，ASR 不够清楚，但能支持勇哥的判断：若真都好赚钱，不应有这么多低价转让或长期空置。

## Yongge Question Order

- 先确认基本沟通、城市层级和是否本地：要求说普通话，问周口、市区/县城/乡镇，确认是县城、本地人。
- 先问目标铺硬条件：面积、一楼150平方米、是否整栋租、二三楼能否用、年租金15万元、月租金12500元、付款方式、一年一付、押金/中介费/转让费、转让费35万元。
- 再问目标铺合理性：位置是否很好、转让费能否谈、为什么现在不好谈、目标铺是不是刚开始转让。
- 转向现有店：问他是不是就在隔壁同街做面馆、准备接下来做什么、为什么现有面馆不继续开、开了多久、一天卖多少钱、离目标铺多远、现店面积多大。
- 要求现场看现店：看门头、格局、楼梯、厨房、堂食区和桌数，用现场观察判断“格局不好”是否真成立。
- 用同街对比逼问逻辑：自己的店租金5000元/月、可能只能转10万，却要去50米外接12500元/月且转让费35万的铺子；问老板是不是把生意不好归因于面积或门头。
- 追问准备做什么：老板没有明确品类后，继续问是否想做冒菜、是自营还是加盟、去哪里学、学哪家、准备花多少钱。
- 做目标铺开店预算：问装修是否还要装、设备和首批食材、准备几个人、当地人工、水电，再按毛利算保本点。
- 追问家庭和分工动机：问儿子年龄、是否上学、几个孩子，确认老板真正动机之一是想把原店交给儿子，夫妻去新店做，而不是已经论证出新店更赚钱。
- 现场看街区：让老板360度看周边，问是否县城中心、客流从哪里来、是否只有这一条平行街、两边是不是餐饮、什么时候人多。
- 追问周边数据：问周边餐饮每天卖多少钱、数据从哪里来、是否现在数据还是节假日数据、是否蹲点；再问各店品类、人员配置，并确认现店是否几乎是整条街生意最差。
- 最后用矛盾反证：如果周边店都1000-3000元/日、租金才5000-6000元/月，为什么还有多家在转、为什么有铺子空一年半没人接。

## Calculation Logic

- 原店租金：6万元/年 = 5000元/月，按30天约167元/日。原店流水约500-800元/日，按55%毛利估算，日毛利约275-440元，月毛利约8250-13200元；扣掉租金后空间很小，还未充分覆盖人工、水电和家庭机会成本。
- 目标铺租金：15万元/年 = 12500元/月，约417元/日。仅租金就是原店的2.5倍，且位置只差约50米，不能假设客流会发生质变。
- 目标铺人工：4人 x 3500元/月 = 14000元/月。
- 目标铺水电：勇哥按整栋铺子估4000元/月，二楼三楼是否使用不清，但成本仍要承担。
- 目标铺月固定成本：租金12500 + 人工14000 + 水电4000 = 30500元/月。
- 保本流水：按综合毛利55%，30500 / 55% = 约55455元/月；按30天折算约1848元/日。勇哥口径取约1800元/日达到保本。
- 这个保本点还没有摊销35万元转让费，也没有算加盟费/学技术费、额外改造设备和老板家庭实际收入要求。因此“1800元/日保本”是偏保守的门槛，不是赚钱目标。
- 若再加入转让费，风险会明显放大：35万元是一次性沉没成本，即便按36个月摊销也接近9722元/月，相当于每天再增加约324元固定压力；但勇哥在现场主要用“不含转费也要1800元/日”来说明已不稳。

## Diagnosis

- 主要问题不是选址，而是错误归因。现店和目标铺同街相隔约50米，客流环境基本一致；现店生意差不能简单归因于面积、楼梯、厨房或门头。
- 老板没有诊断现店为什么卖不动。周边店据称能做到1000-3000元/日，现店只有约500-800元/日，说明更可能是产品、味道、品类组合、门头表达、转化率或运营能力问题。
- 目标铺会显著抬高固定成本。原店月租5000元，目标铺月租12500元，还要背35万元转让费和三层整租压力；如果现有能力在低租店都守不住，高租大铺更难守。
- 老板有“换大位置/换品类/加盟或学技术就能解决”的倾向，但品类没有选定，技术来源没有核实，当地冒菜认知也不清楚，属于先接铺后找项目的高风险顺序。
- 周边数据没有验证。老板知道一些店的流水，但来源多是别人说的，没有蹲点统计；同时街上有多家转让和长期空铺，与“周边都很好卖”存在冲突。
- 家庭分工动机干扰了商业判断。想把原店交给儿子、自己夫妻再开一个新店，是家庭安排，不等于新铺模型成立。

## Recommendation Pattern

- 明确不接目标铺。35万元转让费在县城餐饮铺上缺乏足够证据支撑，且现有能力和数据都不足以证明能做到1800元/日以上。
- 不要为了门头更大、面积更大、同街50米的位置差异去承受2.5倍租金和35万元转让费。先解决现店为什么卖不动，再考虑扩张或换址。
- 用现有低租店作为试错底盘。原店租金低、已经有设备和基础客流，比接高价铺更适合做产品调整、菜单测试、米线/面类结构优化或重新调品。
- 让老板或儿子连续统计3天周边店数据。勇哥要求重点统计每天进店人数、品类，而不是只问营业额；统计后再结合客单价、人员、租金判断哪些品类真实有效。
- 周边营业额要用“进店人数 x 客单价”验证，不能只听员工或老板口述。对节假日数据、活动数据、品牌店和非品牌店数据要分开看。
- 若后续想换品类，先确认当地认知、竞品强弱、客流时段、客单价和毛利结构，再决定是否学习技术或加盟；不要先付转让费、先租整栋楼。
- 如果确实要让儿子接店，也应先把原店的产品、动线、出餐、菜单和获客问题查清楚，而不是把问题留给儿子、自己另开一个成本更高的新坑。

## Reusable Heuristics

- 同一条街相隔几十米的换铺，通常不能当作重新选址红利；先证明原店差是位置问题，否则只是把同一个经营问题搬到更贵的壳里。
- 老店卖不动时，优先问“周边同客流店卖多少、我为什么卖得差”，再问“我要不要换位置/换品类/加盟”。
- 大铺不是答案，保本点才是答案。先算租金、人工、水电，再除以毛利率；若不含转让费都保不了本，含转让费只会更危险。
- 县城餐饮铺出现高额转让费时，要用长期空铺、多家转让、周边真实客数来反证价格是否合理；不能只看门头大和位置看起来热闹。
- 周边数据不能只问营业额。更可迁移的调研动作是连续数天统计进店人数、品类、时段、客单和活动，再倒推真实模型。
- 如果老板说“我找不到原因”，勇哥会先逼他回到现店和周边数据，而不是顺着推荐新项目。没有诊断，就不做扩大投入。
- 家庭接班、夫妻分店、想给孩子安排事情，不能替代商业模型。家庭分工可以是目标，但必须服从单店盈利和风险承受能力。
