## User Facts Extracted

- 咨询者在香港，计划把一个内街铺位买下来做餐饮。铺位本来是两个店连在一起，后面还有空间，合计约100平方米。
- 买铺报价约600万港币。勇哥现场确认不是人民币，按当时口径约折合540万人民币。按约100平方米换算，约1076平方英尺，咨询者口径约6000港币/平方英尺。
- 如果只租，ASR 显示每个铺位现租约1.1万港币/月，两个约2.2万港币/月。年租金约26.4万港币，对600万港币买价的粗租金回报率约4.4%，静态租金回本约22.7年，未计空置、维修、税费和资金成本。
- 咨询者想买而不是租，主要理由有两个：做米线需要打通和装修，装修费接近100万港币；香港铺租常见两年加租约20%，担心租下来装修后被加租。
- 房东有出售意愿，咨询者说房东家里几兄弟姐妹在争执，所以价格看起来低。
- 咨询者已有米线相关经验，但不是完全自己深度经营。朋友自己有一家米线店，咨询者估计利润十几二十万港币/月；咨询者和朋友合伙的另一家店约赚2万港币/月，咨询者后面说自己占股三分之一，投了40多万港币，约一周去一次，主要做营销或有事时处理成本等问题。
- 新店计划做类似香港谭仔/小锅米线方向，不做传统过桥米线。咨询者提到香港顾客更习惯厨房做好端出，而不是一堆小碟自己操作。
- 装修和设备改造成本很高。若用燃气，改造成本高，整体装修约100万港币；若用电磁炉，装修可能约80万港币。ASR 对燃气/用电后期费用有噪声，但最终勇哥判断在该场景下用电更合适。
- 若买铺并装修，总投入约700万港币。咨询者和朋友拟一人一半出资；咨询者表示自己能拿出300万港币，但也想负债约三分之一。
- 新店预计请6个人。香港当地服务员约1.5万-1.8万港币/月，部分岗位约2万港币/月，6人整体人工约12万港币/月。
- 勇哥粗按水电等费用约6000港币/月、毛利率约65%测算，日保本流水约7500港币。这个口径应包含租金或买铺后的租金机会成本，否则只算人工和水电，保本线会低于勇哥口播数。
- 咨询者本职是运动培训/田径教练，按他的说法有很多学生，平均学费约200港币/小时。除培训外，也做美股、日股等投资，买铺的钱部分来自这些积累。
- 店铺位置在内街或巷子里，不在主街。咨询者认为外面主街铺位以前卖到3500万、5000万港币，同面积约100平，外面租金约是这个内街铺的6倍，所以内街600万港币像是便宜机会。
- 勇哥让他现场走动后看到：该铺所在巷子一头接近尽头或断头路，另一头出去才是主街；巷内有部分卷帘门关着，右侧多家店关门或空置，咨询者说空铺约两家。
- 周边有茶餐厅、烧烤、日本餐厅、酒吧、外卖店等。部分店很火，但勇哥追问后确认，生意好的几家多是10年以上老店，属于老牌老字号，有指向性消费，不等于巷子自然流量强。
- 咨询者不知道周边茶餐厅、隔壁店、后面店的日流水；只知道周边有哪些店，且自己在附近待了三四年、蹲过接近一周。勇哥抓住这一点，说明他对真实营业能力没有数。
- 咨询者想做学生生意。他提到附近有三所中学，每所约750人，合计约2250名学生，学校到店约350米，并认为学校周边没有餐厅，学生可能要走这条路下来。

## Yongge Question Order

- 先确认地点和基本动作：在香港，准备把两个相连铺位买下来做餐饮，而不是简单租店。
- 追问面积、买价和币种：多少平、卖多少钱、是港币还是人民币，并顺手把600万港币折成人民币约540万，同时问到每平方英尺价格。
- 追问为什么买而不是租：先听咨询者讲装修成本高、两年加租约20%，再复述确认他买铺的底层理由。
- 追问正常租金：每个铺位租多少钱、两个加起来多少，用租金去校验600万港币的购买价格是否真划算。
- 确认品类：准备做米线，进一步问是过桥米线还是小锅/港式习惯的米线，避免只停留在“米线”大类。
- 追既有经营成绩：现在自己或朋友已有几家店、分别是谁的、每月赚多少钱，区分朋友店的高利润和咨询者自己参与的合伙店利润。
- 追新店投入和配置：装修多少钱、买铺加装修共多少钱、准备请几个人、香港人工成本多少、水电燃气怎么选、毛利率多少。
- 当场算保本：用人工、租金/机会成本、水电和毛利率折成日流水，得出约7500港币/天才保本。
- 追咨询者本人能力圈：以前做什么、为什么做餐饮、本职训练业务如何赚钱、买铺的钱从哪里来、能不能一次拿出300万港币、是否要负债。
- 追管理参与度：上一家合伙店谁在管、咨询者多久去一次、占股多少、投了多少钱，判断他是否真具备亲自经营新店的能力。
- 让咨询者360度看现场：先看店门口、内街、茶餐厅、外卖店，再走到主街和巷子另一头，判断这是不是自然流量位置。
- 追周边数据：隔壁和背后的茶餐厅卖多少钱、周围哪些店数据知道、巷子里有哪些业态、哪些是老店、开了多少年。
- 追目标客群假设：学校在哪里、距离多远、学生人数多少、学生是否真的会走到这条内街消费。
- 最后给投资判断：先肯定咨询者本职能力和赚钱能力，再把问题转回投资眼光、铺位流量、资产增贬值和买铺风险。

## Calculation Logic

- 买铺本金：600万港币，ASR 中勇哥折算成人民币约540万。若再加装修约100万港币，新项目初始投入约700万港币。
- 面积单价：约100平方米约等于1076平方英尺，600万港币对应约5570港币/平方英尺；咨询者口头估算约6000港币/平方英尺。
- 租金参照：现租约1.1万港币/月/间，两间约2.2万港币/月，约733港币/天，年租金约26.4万港币。
- 租售比粗算：年租金26.4万 ÷ 买价600万，约4.4%年化租金回报；静态约22.7年回本。这个回报看起来不低，但前提是持续满租且租金不跌，而现场看到的空铺和内街流量会削弱这一前提。
- 新店人工：6人合计约12万港币/月，约4000港币/天。
- 水电等费用：勇哥现场粗按约6000港币/月，约200港币/天。ASR 对燃气和电的相对费用有噪声，但结论是燃气改造贵，电磁炉方案更适合粗算。
- 若把两间铺的租金或买铺后的租金机会成本计入固定成本：人工12万 + 租金机会成本2.2万 + 水电约0.6万 = 约14.8万港币/月，约4933港币/天。
- 按毛利率65%倒推保本流水：4933 ÷ 65% ≈ 7590港币/天，基本对应勇哥口播“7500块钱才能达到盈亏平衡点”。
- 若不计租金机会成本，只算人工12万和水电6000，固定成本约12.6万港币/月，约4200港币/天，保本流水约6460港币/天。但买了铺并不等于铺子免费，做投资判断时仍应把租金机会成本或资金占用算进去。
- 现有合伙店约赚2万港币/月，且咨询者占股三分之一、参与度约每周一次。这个利润规模不足以证明他能亲自操盘一个700万港币投入、日保本约7500港币的新店。
- 学生客群假设：三所中学每所约750人，共约2250人，距离约350米。ASR 没有证明这些学生当前会到内街消费，也没有周边店流水支持，因此不能直接折算成有效客流。

## Diagnosis

- 这是一个开店前的投资/选址拦截案，不是已经亏损后的止损案。核心风险是把“看起来便宜的内街资产”误判成“适合餐饮的高性价比铺位”。
- 600万港币便宜，是相对主街3500万、5000万港币的成交价便宜；但餐饮看的是到店流量、可见度、路径和转化。巷子只差一条街，价值和经营能力可能完全不同。
- 勇哥现场判断该位置本质是巷子或内街，并且一侧接近断头路。它不是顾客自然经过的位置，只有主街才有真实人流。
- 巷内生意好的店多是10年以上老店，靠老字号和强目的性消费生存。新品牌米线不能把老店的排队现象当成自己也能承接的自然流量。
- 空铺和关门卷帘门是价格信号。房东愿意600万港币卖、周边有空铺，并不必然是捡漏，可能是该位置抗风险能力差、租售承压。
- 咨询者掌握的周边数据不够。虽然在附近待了三四年、蹲过约一周，但不知道茶餐厅、隔壁店、背后店一天卖多少，只知道哪些店“很火”或“老店”，还没有完成选址验证。
- 买铺和开店混在一起，风险被放大。买铺要看资产增值、租售比、流动性和贬值风险；开米线要看日流水、毛利、人工和客流。两件事任何一个不成立，700万港币投入都会被拖住。
- 咨询者的本职训练业务和投资能力较强，但餐饮经营参与不深。合伙店每周去一次、占股三分之一、主要做营销或处理问题，不等于能操盘高投入新店的选址、组织、出品和日常管理。
- 这个米线模型的人工压力很高。香港6个人约12万港币/月，保本日流水约7500港币；如果位置没有自然流量，新店很难只靠学生和熟客快速达到这个线。
- 学校客群没有闭环。三所中学和2250名学生只是潜在人数，350米距离、是否顺路、学生午晚餐限制、价格接受度、竞品选择，都没有被真实流水验证。
- 勇哥反对的不是香港买铺，也不是米线本身，而是在内街低价铺上同时下注资产和餐饮经营。他明确认为这个投资眼光没有想透，可能回不了本。

## Recommendation Pattern

- 不建议买这个内街铺位。600万港币买铺加100万港币装修，资金占用太大，且位置不具备足够强的自然流量和资产抗跌能力。
- 不要把“外面卖三四千万、里面只卖600万”理解成捡漏。勇哥的判断是，就差一条巷子，商业价值可以差很多；便宜来自位置弱，而不是一定来自低估。
- 如果只是为了开餐饮，宁愿选外面主街的高租金铺，也不要选内街便宜铺。咨询者问“里面2万、外面8万选哪个”时，勇哥明确倾向选外面8万，因为餐饮先要流量。
- 如果是资产投资，也应买外面更优质的位置。好地段在下行时更能扛，内街弱铺遇到市场变化时可能跌得更快、更难出租或转手。
- 若继续考虑米线新店，先租后买、先验证后装修。至少要拿到周边竞品真实流水、学生客群转化、午晚高峰人流、外卖能力和保本日流水，再谈大投入。
- 不建议为了规避两年加租而直接买铺。装修贵和怕加租，只能说明租赁有风险，不能自动证明买入弱位置是正确答案。
- 咨询者更应继续做自己擅长的运动培训：扩大招生、请教练、自己做营销和管理，把本职业务做大。餐饮和商铺投资可以做，但不要离开能力圈太远。
- 若未来仍要做餐饮投资，咨询者应把自己定位为投资人而非凭感觉选址的人：让专业店长或合伙人承担经营，同时自己必须掌握基本选址和算账框架。

## Reusable Heuristics

- 买铺开店要拆成两张账：一张是资产账，看买价、租售比、空置、流动性、增贬值；一张是经营账，看日流水、毛利、人工、租金机会成本和保本线。
- “便宜”不是选址理由。主街和内街只隔一条巷子，客流、曝光和抗跌能力可能差几倍；餐饮不能只按平方米或平方英尺单价判断。
- 内街老店生意好，不代表新店能活。老字号靠目的性消费，新店靠自然流量和转化；两者不是同一种能力。
- 看到空铺、卷帘门、断头路，要先问为什么空、空了多久、原店为何关、旁边真实流水多少，而不是先问价格有多低。
- 竞品流水必须问实数。只说“很火”“排队”“老店”不够，至少要估算日流水、客单、翻台、外卖单量和高峰时段。
- 学校、社区、写字楼只是潜在客群，不是有效客流。要验证距离、路径、是否顺路、用餐时间、消费能力、是否已有替代选择。
- 买了铺也要算租金机会成本。自有物业不是免费门店，如果不把可出租租金或资金成本算入经营账，会高估餐饮项目盈利。
- 高人工城市开餐饮，保本线会被人工快速推高。6个人12万港币/月时，即使租金不高，日流水也要达到数千港币才有讨论空间。
- 合伙店赚钱不等于自己能复制。要看自己占股多少、是否日常管理、利润归属口径、是否靠朋友能力和老店基础，而不是只看“已有餐饮经验”。
- 为了躲避未来加租而买铺，只适合强位置、强模型、强现金流。弱位置买下来，可能把租赁风险变成资产套牢风险。
- 餐饮选址如果在“里面2万租金”和“外面8万租金”之间选择，不能机械选便宜。若外面有稳定人流和更强转化，高租金反而可能是更可控的经营成本。
- 投资要回到能力圈。餐饮不是不能投，但不熟悉选址、管理和产品的人，应该先低成本验证或找能经营的人，而不是用大额买铺来证明决心。
