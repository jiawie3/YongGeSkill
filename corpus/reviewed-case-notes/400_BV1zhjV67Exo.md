## User Facts Extracted

- 咨询者不是老板，是店内运营，约38岁，餐饮运营经验约1-2年；老板平时没时间管店，另有店长和厨师长。老板此前未开过餐饮店，ASR称原来做配送类业务。
- 店在咸阳郊区某商场内，ASR听作“新鲜五月/奉行五月广场”，商场名不可靠。店位于5楼，4楼似为主餐饮区；店在五楼角落/端头位置，平时客流弱，周末和节假日客流较多。
- 品类为陕西菜，夹杂融合菜、家常菜/川湘类菜；店本质是中餐馆。周边有二十多家餐饮，竞品多为180-300平小店，日流水约1.5-1.8万，节假日能排队。
- 门店面积约1500平，两层：一楼约800-900平、40多张台；二楼8个包间。商场无独立电梯，包间只能从店内上楼。
- 店内43人，其中3名演艺人员；有舞台，古风舞蹈约半小时一场。装修/装置投入很重，个别装置ASR提到十几万、二十多万。
- 投入总额约456万，后面对话又反复说“400多万砸进去”。勇哥把这视为沉没成本，不能作为继续亏损的理由。
- 开店时间为2023年9月，按对话口径已经营接近两年。
- 正常流水：周一到周五约1万多/日，周末约2-3万/日；月流水最高做过69万，但勇哥要求不要看最高值，正常月流水约52-53万，折合日均约1.7万。
- 食材口径毛利率约61%。勇哥提醒这只是食材毛利，包装、损耗等若未计入，会高估真实毛利率。
- 月固定和半固定开支：租金加物业约11万/月，折合约3600元/日；人工约23万/月，折合约7600元/日；水电气约4万多/月，ASR中为“四万零八十左右”，折合约1333元/日；广告/运营推广费约3.5万/月，折合约1200元/日；两个员工宿舍6000元/月，折合约200元/日；维修、包装、损耗等其他费用ASR粗糙，约5万多/月，折合约1800元/日。
- 低价活动：做过39.9元烤鱼套餐，套餐含约1斤2两烤鱼、6个配菜、大桶饮料、3碗米饭、2份火鸡面，定位2-3人餐。鱼成本约25-26元，整套核算每套约亏3元。
- 低价券数据 ASR 前后有冲突：曾说“卖了6000多套”，后又说“卖了2万多张，上个月核销6000多”。较可信整理为：售出2万多张券，上月核销6000多，之后仍每天核销约100-160张、节假日约200-270张；6月3/4日前后停止继续售卖。90%顾客只吃烤鱼套餐，几乎不加点其他产品。

## Yongge Question Order

- 先确认来电者身份：是不是老板；如果不是，明确其职位、年龄、运营经验、老板是否参与管理，以及门店当前到底遇到什么问题。
- 追基本盘：位置、品类、面积、周边客群、平日与周末生意差异。
- 追流水时先问“现在一天卖多少”，再问“一个月营业额多少”；当对方报最高月69万时，立即打断，改问正常月流水，得到52-53万。
- 按损益顺序追数字：毛利率、租金物业、在岗人数、人工、演艺人数、水电气、广告投流、宿舍、维修包装损耗等其他费用。
- 现场把月口径换成日口径：租金3600/日、人工7600/日、水电约1333/日、推广约1200/日、宿舍约200/日、其他约1800/日，由此推到每天亏损和每月亏损。
- 追活动是否越做越亏：问为什么做赔本套餐、卖了多少、核销多少、每套亏多少、是否带动其他消费。确认低价活动只带来薅羊毛客，未形成加点。
- 追经营年限和咨询目标：什么时候开的、平时流水、咨询者到底想解决什么。
- 转为视频巡店：看楼层、动线、右侧陈列、舞台、两层结构、桌数、包间、展示面、竞品面积与竞品流水、五一等大节是否排队。
- 追可调整性：问店能不能变小、能否分割、能否店中店、能否和商场谈。再问老板背景、老板与运营的关系、推广是谁做、运营具体做什么。
- 最后才给诊断和动作：先定“面积太大”这个硬性矛盾，再处理低价套餐、包间、演艺、推广和沉没成本。

## Calculation Logic

- 正常月流水约52-53万，按30天折合日均约1.73-1.77万。
- 食材毛利率约61%，日毛利约1.05-1.08万。勇哥默认先用食材毛利算，但后面强调真实毛利应把包装、损耗等纳入。
- 已确认日成本：
  - 租金物业：约11万/月，约3600元/日。
  - 人工：约23万/月，约7600元/日。
  - 水电气：约4万多/月，约1333元/日。
  - 广告/运营推广：约3.5万/月，约1200元/日。
  - 员工宿舍：6000元/月，约200元/日。
  - 维修、包装、损耗等其他费用：ASR粗糙，约5万多/月，约1800元/日。
- 勇哥先用毛利扣租金、人工、水电：约1.06万 - 3600 - 7600 - 1333，得到每天亏约2000元，月亏约6万。
- 再把推广、宿舍、其他费用补进去，水电+推广+宿舍+其他约4533元/日；整体固定/半固定开支约3600 + 7600 + 4533 = 15733元/日。
- 以61%毛利率倒推盈亏平衡流水：15733 / 0.61 ≈ 2.58万/日，勇哥口径为约2.6万/日。
- 当前日均约1.7万，离2.6万保本点差约9000元/日；按61%毛利贡献，收入缺口对应约5500元/日利润缺口，所以月亏约15万的结论成立。
- 勇哥给的降本目标是示范性口径：房租若从3600/日谈到1500/日，约省2000元/日；人工7600/日砍掉一半，约省3800-4000元/日；杂费4533/日再压3000元左右，亏损模型才可能反转。

## Diagnosis

- 核心不是菜品短期不好吃，而是店型模型错了：1500平商场中餐店，对应租金、人工、空场感、动线和管理成本都太重，现有商场客流与品类能力撑不起2.6万/日的保本流水。
- 面积错配导致人效、坪效都低：竞品180-300平能做1.5-1.8万/日，本店1500平日均也只有约1.7万，多出来的约1300平没有创造相应营业额，却持续吞掉房租和人工。
- 二楼包间在商场场景里价值低：商务/宴请通常需要更晚结束和更私密便利的到达方式，商场10点打烊、无独立电梯，只能支撑少量家庭消费，难以承担包间面积成本。
- 演艺是老板自我想象，不是顾客核心需求。商场餐饮顾客主要来吃饭，不是为看几秒到一分钟舞蹈而来；演艺人员每天约600元、月约1.7万，未证明能带来两三千元日增量。
- 低价套餐策略错误：39.9烤鱼套餐把主菜、饭、面、饮料、配菜都包完，顾客没有加点空间；90%只消费套餐，活动越核销越亏，还没有沉淀正常客单。
- 运营能力和老板认知不足：老板没餐饮经验，却跟风现象级单店烤鱼模型；运营也没有识别“单品店爆火”和“大型中餐店做赔本套餐”之间的模型差异。
- 400多万装修投入、十几万押金不是继续亏损的理由。勇哥把它定义为沉没成本：现在应看未来每月能不能赚钱，而不是为了过去投入继续扛每月15万亏损。

## Recommendation Pattern

- 第一优先级：让老板和商场谈面积、租金、分割、分租或退租重签。只要面积问题不解决，其他推广动作都只是拖延。
- 如果短期无法缩小租赁面积，先在经营上“缩小可见营业面”：平日只开一楼一部分，例如40多张台先放约20张，用屏风遮挡空区；二楼包间平时先不用，大节假日再全开。
- 用有限座位制造羊群效应。周末或节假日要让店看起来有人、必要时能排队；长期大面积空场会让顾客判断这家店不好吃。
- 停止低价赔本套餐，不再用39.9这类打不透的大店引流。改做常规、有品质、吃得饱且不出毛病的套餐，价格不必极低，关键是体验可接受且毛利成立。
- 砍掉演艺和无效人工，按照缩小后的营业面重排岗位；演艺费用若不能证明带来足额新增营业额，就不应保留。
- 重新算真实毛利，把包装、损耗、杂费计入，不要只看食材毛利率。老板需要逐项列出哪些可少、哪些可减、哪些必须保留。
- 若商场不愿协商、面积和租金无法下降，勇哥建议宁可关店/退掉重开。每月亏15万时，十几万押金或已投入装修不应绑架决策。

## Reusable Heuristics

- 咨询者报“最高流水”时，要立刻拉回“正常月流水”和“平日/周末日流水”；餐饮诊断用常态数据，不用峰值安慰自己。
- 大店先算保本流水：日固定成本 / 毛利率 = 保本日流水。若现有商圈客流和竞品表现撑不起这个数，优先处理面积和租金，不要先谈营销。
- 坪效要和同楼层竞品横向比。若1500平只卖得和180-300平竞品差不多，多出来的面积就是亏损源。
- 商场中餐包间不能按社会餐饮包间想象价值，必须验证独立到达、营业时长、宴请需求和使用率。
- 活动引流要看“是否带来加点和复购”，不能只看核销量。套餐把顾客吃饱且单套亏钱，就会把门店变成低价券核销场。
- 演艺、装置、装修都要问“给顾客创造什么购买理由、带来多少新增营业额”。无法量化增量时，先按成本项处理。
- 沉没成本不参与继续经营判断。已经投入的装修和押金，只能作为谈判筹码或损失确认，不能成为每月继续亏损的理由。
- 当主要矛盾是结构性成本，推荐顺序是：缩面积/谈租金 -> 缩营业面和人工 -> 修正毛利口径 -> 停止无效低价引流 -> 再做正常推广。
